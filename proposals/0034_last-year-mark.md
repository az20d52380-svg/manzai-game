# イベント会話提案 0034「去年の印」v1（2026-07-06・ドラマ主軸・レッドチーム済）

- 日付: 2026-07-06 ／ 数値は全て【仮】 ／ 既存ファイルは読むだけ ／ トーン＝ドラマ主軸
- 種別: **非選択の「会話イベント」**（大会前夜の別バリエーション）。0010-0031 のパワプロ型A/B選択とは別カテゴリ＝選択肢・効果ロールを持たない一場面会話。

## 狙い（どの薄いスロットを埋めるか）

大会前夜（`weeksLeft==1`）は本作最大の感情ビートの直前だが、ここに立つ会話が薄い。現状の前夜は実質**メンタル帯（#28 眠れない夜＝自動）**と**0010「前夜の一本」＝詰める/整える選択**しかない。0010 は「緊張の詰め／整え」を核にした選択で前夜の題材を直撃している。

本案は**同じ前夜スロットに、緊張の詰め/整えを核にしない別ビート**を差す。核は**ラダー（同じ回戦で足踏み）**。ただし——

- **既存territoryとの関係**: 「同回戦・経過年で壁を名指す」ラダーは**既に正典が三か所ある**——(1) dialogue_design §4「同回戦3年連続」（敗退後の谷口/俺の掛け合い・壁の擬人化）、(2) 0003 の `sameWallStreak` ladder2/3/4（**本編機能・多年のみ**）、(3) fable :166-170（週頭独白／年次のラダー行）。本案はこれらと**題材（同回戦足踏み）は同じだが、チャネルと質感が違う**。§4・0003 は**敗退直後の声（電車/リザルト）**、fable は**週頭独白・年次一言**。本案は**前夜・楽屋・物**——声でなく**去年の楽屋に残った物**でラダーを立てる。埋める穴はこの一点＝「敗退の直後でなく、次に挑む前夜に、同じ場所へ戻ることを事実で置く」。
- **§3-B の別実装**: 壁を抽象的連敗数でなく回戦名＋経過年で名指す（fable §3-B）を、平常スロットの一行心の声でも敗退後の声でもなく、**楽屋の固有プロップ**で身体化する。
- **§4 の擬人化行は再演しない**: 壁を「名前で呼ぶ／向こうが俺らを覚えた」の擬人化は §4／0003 ladder の正典＝**本案では言い換えない**（近似行フォーク回避・0003 レッドチームB-3 と同じ規律）。本案の物＝**壁の擬人化とは別系統の「正の字」**。

## 発火条件（実フック）

- **一次フック**: `nextMilestone()`（`WeekMainView.swift:407-415` 相当）の `weeksLeft == 1`。育成 monoBox で到達しうる最小＝1（大会週 weeksLeft==0 は `tournamentDecision`＝別画面で育成イベント判定は走らない）。0004/0010 で確認済みの「前夜」。
- **格ゲート（非決勝）**: 明日の大会が**決勝でない**回戦のときのみ。決勝前夜は山場側に譲る。格の判定は `nextMilestone().name` を `config.calendar` のラベルと突合してUI側で導出（新フィールド不要・0010 と同じ手筋）。
- **ラダーゲート（本案固有・0003 流用）**: **`sameWallStreak >= 3`**（明日と同じ回戦に過去到達して敗退・3年以上足踏み中）。**新フィールドは起こさない**——0003 が既に設計した `lastLostStageName:String?` / `sameWallStreak:Int`（回戦名の文字列 keying・`GameSession.swift:148-152` の `lossStreak` 更新と同一トランザクションで加算）を**唯一の正典として流用**する。`lossStreak`（連敗数）や `justPassedStage` は回戦の同一性を持たないため使えないのは下書き指摘どおりだが、その受け皿は**既に 0003 にある**。二つ目の履歴フィールド（`stageHistory` 等）を新設すると同回戦カウンタが二重化＝正典フォークになるので**しない**。
- **【本編機能・MVP非発火】**: `sameWallStreak >= 3` は**多年でのみ真になる**。MVP は `GameSession.swift:36 year=1`＝各回戦名は年内に一度しか出ないため、同回戦の再会（`sameWallStreak>=2`）は起きない（0003 が確定済み・レッドチームB-2）。**よって本案は 0003 ladder と同じ本編（多年）機能で、MVPでは発火しない＝欠陥ではない**。前夜の穴をMVPで埋めるのは 0010（選択）／#28（自動）側。本案はその多年版の彩り。
- **効果**: **なし（純フレーバー）**。数値はセリフで言わない。
- **【golden安全の芯】** 非選択＝**確定発火・抽選ロールなし・効果ゼロ**。第一根拠＝**イベント層は golden 対象外・既定OFF**（0028 が `CareerGoldenTests` の正典コメントで確認した通り、シーンも効果も auto-sim では発火しない）。第二の保険＝発火は決定的フラグ（`weeksLeft`／格／`sameWallStreak`）のみで**乱数を1ビットも引かない**。ゆえに**3年golden不変・`tools/*.py` 正典同期も不要**。ここが選択肢イベントとの最大の違い＝実装が軽い。**効果を付けたい／抽選で頻度を絞りたくなった瞬間に消費順の正典に触れる＝その時は必ず止めて相談**（CLAUDE §A）。

## イベント会話本文

**セットアップ（地の文・俺視点・標準語）**

```
明日で、この三回戦は三度目だ。三年、同じ高さで足踏みしている。楽屋も、去年と同じ部屋をあてがわれた。
去年、出番を待つあいだに、俺は壁の隅へ鉛筆で正の字を書いていた。前に何組控えているか、数えるためだ。
四画まで書いたところで名前を呼ばれて、五画目は書かなかった。
その四画が、まだ残っている。この楽屋の壁は、去年から誰も塗り替えていない。
```

谷口「去年のやつ、まだ消えてへんな」

```
谷口は鉛筆を手に取ると、四画の横に、間を空けて一画だけ足した。順番の数とは、続きになっていない位置だった。
何も言わずに、鉛筆を元の場所へ戻した。
```

俺「それは順番待ちの数だ。続きにするなら、四画の隣じゃない」

谷口「順番の数は、明日で用済みや。俺が足したんは、そっちとは別に数えとる。……来年も同じ部屋やったら、続きから足せるな」

**谷口以外の相方版（標準語ベース・別プール／地の文の所作は共通）**

相方「去年の、まだ残ってるね」

（同・地の文：相方が鉛筆で一画だけ足す）

俺「それは順番待ちの数だ。続きにするなら、四画の隣じゃない」

相方「順番の分は、明日で消えるよ。ぼくが足したのは、別に数えてる。……来年も同じ部屋なら、続きから足せるね」

**相性帯での温度差（会話は差し替えず、所作の地の文だけ微調整・任意）**
- 0-7（他人行儀）: 一画を足す前に少し間が空く（「鉛筆を手に取ると」→「壁の前で一度止まってから」）。
- 8-14: 上記本文どおり。
- 15-20（ツーカー）: 谷口の一言目「去年のやつ、まだ消えてへんな」を省き、**無言で一画だけ足して**から最後の一言に飛ばす（阿吽の谷口ほど言葉が減る）。※compat上限20＝阿吽21+は出さない。

*（会話は話者ごと1セリフずつの通常送り・「／」詰めなし。ボケ＝オチ/フリ/ネタ本文ゼロ＝笑いは地の文にのみ。乾いた一滴は谷口の最後「来年も同じ部屋やったら、続きから足せるな」のデッドパン1箇所だけ＝真顔で事実を並置・自己言及ゼロ・足踏みを泣き言でなく"数え方の段取り"に逃がす。俺＝標準語・計算屋で「それは順番待ちの数だ」と物の意味を差引で正すだけ・自己憐憫も警句締めもなし。熱・執念は俺に持たせず谷口の所作へ寄せる＝§3-B。谷口＝関西弁一貫、相方版＝標準語で関西弁を混ぜない。）*

## 固有プロップの節度

- **1個＝去年と同じ楽屋の壁に、去年の自分が鉛筆で書いた順番待ちの正の字（四画）**。塗り替えられずまだ残っている。定番プロップ〈天井の染み・冷めたコーヒー・小銭〉ではない。**この楽屋・この足踏み固有の物**として置き、他イベントで使い回さない。
- **谷口の一画＝置換不能の所作**: 谷口は緊張の話をせず、四画の**隣ではなく間を空けた位置**に一画だけ足す＝「順番待ちとは別に数えている」を**位置で示し、感情を言語化しない**（氷山／§3-B）。谷口の本音は「別に数えとる」「来年も…続きから足せる」の**段取りの言い回し**までで止め、悔しさ・執念は喋らせない。
- **§4擬人化との非重複**: 「壁を名前で呼ぶ／向こうが顔を覚えた」の擬人化行（§4・0003 ladder・fable :167）は**使わない**。本案の物は正の字＝別系統。

## どこから学んだか（型の出典・任意URL）

学んだのは**構造・型のみ**。既存作品のセリフ・本文は転載しない。

- **§3-B「固有の壁を回戦名＋経過年で名指す」**（自作正典）。抽象的連敗数でなく「三回戦・三年」で足踏みを事実化する型を、平常心の声でも敗退後の声でもなく前夜の一場面に移した。`proposals/fable_kickoff_drama_allin_v1.md` §3-B。
- **確定イベントの二層（節目＝必ず起きる／彩り＝ランダム）**。前夜は運任せでなく「同じ場所に戻る」を確定で置く節目。パワプロ「サクセス」確定イベント構造。https://ja.wikipedia.org/wiki/実況パワフルプロ野球_サクセスモード
- **下積みの非対称・世界の事実の積み方（声は移植しない）**。「同じ高さで足踏み」を感想でなく事実（塗り替えられない壁・去年の四画）で置く。又吉直樹『火花』の世界＝構造のみ抽出、「僕」・関西の思弁・比喩の連射は移植しない。https://ja.wikipedia.org/wiki/火花_(小説)
- **氷山理論・言い落とし（志賀直哉／カーヴァー）と生活の物（向田邦子）**。心情を説明せず「四画の隣に間を空けて一画足す」という**所作1つ**に意味を背負わせ、判断を受け手に渡す。谷口の本音は言語化しない。`proposals/0009`（感情でなく物）。https://toyokeizai.net/articles/-/717910

## 実装イメージ（既存の会話/イベント機構への載せ方）

- **種別**: 選択肢なしの一場面会話＝**DialogueData/WeekMainView の会話送り表示系にそのまま載る**。0010-0031 の「2択オーバーレイ＋効果パッチ」基盤は**不要**＝必要なのは話者切替の会話送り（俺/谷口/相方）＝**順送りの非選択シーンランタイム**（0015/0028 が未実装と指摘した器。選択肢なしぶん 0024 型より単純）。
- **golden/乱数消費順**: **触れない**。確定発火・抽選なし・効果ゼロ＝runner の RNG を一切引かない＝3年ビット一致は不変で `tools/*.py` 正典同期も不要（第一根拠＝イベント層 golden-OFF・0028）。**効果を付けたくなった／抽選で頻度を絞りたくなった瞬間に消費順の正典に触れる＝その時は必ず止めて相談**（CLAUDE §A）。第一弾は**純フレーバー（効果ゼロ）**推奨＝完全にgolden安全。
- **ゲート材料は 0003 流用（新設しない）**: 「同じ三回戦に過去到達・ラダー3-4年目」は 0003 の `lastLostStageName`/`sameWallStreak` で判定（回戦名の文字列 keying・`GameSession:148-152` と同一トランザクション更新）。**本案独自の新フィールドは起こさない**（同回戦カウンタの二重化＝正典フォーク回避）。`sameWallStreak` は RNG 非消費＝golden不変（0003 と同じ置き場の思想）。**本編セーブへの永続化は多年到達時に 0003 側で別途設計**（MVP不要）。
- **「同じ部屋」の同一性は未モデル**: 地の文の「去年と同じ部屋をあてがわれた」は room 割当を GameCore が追跡していない叙述上の意匠。発火の実体は `sameWallStreak` にのみ乗せ、部屋の同一性は決定的演出（RNG非消費）として扱う。
- **反復＝動的テンプレ**: 同一キャリアで同じ回戦に足踏みするたび再発し得る（3年目・4年目…）。むしろ**足踏みが続くほど印が増える**のが芯なので、固定文の反復は「四画→五画→…と印が増える」動的テンプレ（`sameWallStreak` をそのまま画数に写す）で吸収（`proposals/0001` 直近回避と併用）。
- **目視**: UIは `swift test` で検証不可＝**多年到達（`sameWallStreak>=3`）を作れる本編ビルドで simulator 目視**（CLAUDE §D-10）。MVP（1年）では発火しないため MVP simulator では確認不能＝本編工程で確認。

## リスク・注意

- **【本編機能・MVP非発火】** 発火核が多年でしか真にならない（`year=1`）。0003 ladder と同じ層＝MVP検証で動作しないのは欠陥ではない。前夜の穴のMVP充足は 0010／#28 に委ね、本案は多年版の彩りとして分離。
- **【正典フォーク回避】** ゲートは 0003 の `sameWallStreak` を流用し、擬人化行は §4 を唯一の正典として**再演しない**。本案が独自に同回戦カウンタや壁の擬人化近似行を持つと、片方だけ直った時に文言・条件が割れる。
- **【前夜スロット過密】** 同じ `weeksLeft==1` に #28（自動）・0010（格の高い大会の選択）・本案（非決勝の足踏み・多年）が同居し得る。本案は**非決勝×同回戦足踏み3-4年目**に厳しくゲートして薄く出す。0010（格の高い大会）とは通常衝突しない（本案は格を外す）。万一重なるビルドでは相互排他 or 優先を発火表で明示。

- **【プロップの節度】** 「壁に鉛筆で正の字」は**この楽屋固有の小さな物**として置く（定番プロップ不使用）。落書き感・天丼化を避け、**塗り替えられない安劇場の壁**という質感の内側に留める。囚人が日数を刻む型の常套に寄らないよう、意味は「去年の自分の順番待ち＋谷口が別に数える一画」に固定し、他イベントで使い回さない。
- **【体感を数値で直さない】** 足踏みの重さは**印・所作**で背負わせ、効果（数値）で押し付けない。純フレーバー推奨（ui_design §7-B・0009）。
- **【トーン自己点検】** 感情語（悲しい/悔しい/惨め/嬉しい）不使用＝感情は物（塗り替えられない壁／去年の四画／間を空けた谷口の一画／来年も同じ部屋）に背負わせた。デッドパンの一滴は谷口の最後1箇所のみ（真顔の事実並置・自己言及ゼロ）。俺＝標準語・計算屋（「それは順番待ちの数だ」と物の意味を差引で正すだけ・自己憐憫も警句締めもなし）。熱・執念は俺でなく谷口の所作へ寄せた（§3-B）。谷口＝関西弁一貫、相方版＝標準語で関西弁を混ぜない。第三者は登場しない（関西弁を移す先なし）。会話は話者ごと1セリフ・「／」詰めなし。ボケ（オチ/フリ/ネタ本文/パンチライン）ゼロ＝笑いは地の文のみ。固有名は架空（回戦名は一般語の「三回戦」・大会固有名は出さない）・数値は【仮】・軸名や数値はセリフに出さない（正の字の画数はゲーム数値でなくプロップ描写）。既存作品（火花/べしゃり暮らし/M-1実在ネタ）とのセリフ一致・酷似なし（各行照合）。§4擬人化行・fable隣室ネタ回し・#28・0010・0027 とも別プロップ／別ビートを確認。
