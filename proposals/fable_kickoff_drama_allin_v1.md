# 漫才師コンビ育成SLG ―― 声(voice)の較正キックオフ

## 0. 姿勢

これは量産の依頼ではなく、**較正**の依頼だ。目標は「たくさん書く」ことではなく、**後で別の量産用の書き手が真似られる"金型"を、少数精鋭で正確に彫る**こと。

- 全在庫を書こうとするな。**型を言語化 → 各スロットに数本だけ丁寧に → 自己採点 → 広げる**の順で進む。
- 1本ごとに「基調に合っているか」を自分で採点する。浅い埋め草を100本書くのが最大の失敗。
- 迷ったら本数を減らし質を上げる。**新規本文は全スロット合計で概ね60〜90本を上限**とし、超えそうなら§8の優先順位の上位だけ残して止めてよい(下記の各スロット本数は"天井"であってノルマではない)。

## 1. 役割と成果物

あなたはこの作品の"声(voice)"を確定させる主筆だ。成果物は2枚。

- **【A】voice_bible_v0.md** ―― 文体バイブル。声の「型」を言語化した金型(ルール＋○×チェックリスト＋アンチパターン集＋較正ギャラリー)。
- **【B】voice_corpus_v0.md** ―― 技法タグ付きの gold seed。各スロットに少数精鋭の本文を、良い例と悪い例の対で置く見本集。

**あなたは金型を彫る。別の書き手が鋳込む。** 全在庫は書かない。この2枚が量産のfew-shot基準になる。
両ファイル冒頭に必ず記す:「日付 / 位置づけ / 数値は全て【仮】 / これは"種"であり最終確定は実機目視」。末尾トレーラは不要(ID・重み・整形は後工程の仕事)。

## 2. 芯(不変・絶対の一線)

1. **ネタ本文を一文字も書かない。** 笑いはネタ本文でなく、**地の文(心の声 / 場面描写 / 審査講評)にのみ**宿らせる。ゲームが笑いを取りにいった瞬間に寒くなる。
   - 機械化チェック:**「この一行は、ネタのボケ・フリ・前提・オチ・パンチラインのいずれかを一文字でも本文化していないか?(＝観客に見せたら笑いを取りにいく台詞になっていないか)→一つでも該当すれば没。」**
   - 例外:**審査講評が『オチ / ボケ / テンポ』を批評語として言及する**のは可(それ自体はネタ本文ではない)。禁止は"笑いを取りにいく台詞本文"のみ。
   - **gold seedのどの一本も、ネタのフリ / ボケを含まない**(相方の軽口も含めて)。
2. **固有名は全て架空。数値は全て【仮】。**
3. **既存作品(火花 / べしゃり暮らし / M-1実在ネタ等)のセリフのコピペや酷似は厳禁。** registerが汚染される。
4. **会話は話者ごとに1セリフずつ送る**(「／」で詰めない)。詰めると本番のタップ送りの"間"が死ぬ。
5. **相方依存の声:** 谷口=関西弁で一貫 / 谷口以外の相方=標準語ベース。**俺の地の文・一人称は常に標準語**(「言うた」→「言った」)。

## 3. 基調(ドラマ主軸)と"アンチ寒さ"則

**基調はドラマ。ユーモアは従・薬味。**

### 3-A. 芯の原則
- **感情語(悲しい / 悔しい / 惨め / 嬉しい)を書かない。** 感情が起きた瞬間の**「場所の物」(残高・音・所作・小道具)**に背負わせる。名指しした感情は押し付けで受け手が引く。物に語らせると受け手が自分で温度を埋める。
- **ユーモアは二本立ての乾いた一滴のみ:**
  - (a)**自嘲**(自己憐憫に転ばない乾いた自己言及)/(b)**デッドパン**(真顔で事実を並置・自己言及ゼロ・そのズレから苦笑が漏れる)。
  - 自嘲だけ=湿った泣き言、デッドパンだけ=達観キャラの演技。2種を使い分ける。
  - **頻度の分母は"プレイヤーが実際に連続で引く心の声の並び"。** default系スロットでも6本中1本以下に抑える。seedでは各スロットに一滴入りを**最大1本だけ**置き、残りは無味のドラマ地の文で埋めて"1/6が視認できる並び"を作る。
- **固有/定番の原理(可否の判定基準):**
  - **固有=その稽古場 / 相方 / 客席の固有名詞・履歴に紐づき、他コンビの物語では成立しない物**(例:相方が予約表に書く時の癖字、客席42脚のうち割れた9番)。
  - **定番=どのコンビにも置換可能な物**(コーヒー・蛍光灯・小銭・財布・家賃・染み)。**置換可能なら没。**「少しズレた物」とはこの置換不能性のこと。
  - 物は**1つだけ**選ぶ。

### 3-B. 声のロック
- **俺**=標準語・計算屋・ツッコミ体質・心配性。**ただし自己憐憫もキザ(うまいことを言う計算高さの誇示)も禁物。** 突き放すが、警句・教訓・アフォリズムで締めない。計算屋の目は差引を数えるためで、切れ味を見せるためではない。熱を俺に持たせない。
- **熱・執念・壁の事件化は「俺」でなく谷口・支配人など他者声へ寄せる**(声を固定トーンで分離)。
- **谷口(相方)の本音は言語化しない。**「壁を名前で呼ぶ」「三行目だけ二回読み返す」など**事実1つで置く**。**発話も§2-1の対象**――谷口・相方は生活・段取り・観測・軽い事実だけ喋り、笑いを取りにいく台詞(フリ / ボケ / 天丼のボケ側)は書かない。
- **固有の壁を名前で:**抽象的な連敗数でなく**回戦名＋経過年**で出す(「また二回戦。三年、同じところにいる」)。乾いた事実がドラマを語る。

### 3-C. 観測レンズ(平常の心の声の6つの目)
損益 / 心配 / 谷口観測(事実だけ・本音は言語化しない) / 上向き(抑制して温度だけ) / デッドパンの一滴(低頻度) / 自嘲の一滴(低頻度)。

## 4. 内蔵する本技法(源泉ごとの操作手順)

**§4は主武器ではない。主はドラマ(人物の事件と、感情を物へ転写すること)。技法は"書いた後の検算用チェック"であって、技法を成立させるために一行を書くな。技法名が先に立った行・文法規定をなぞっただけの行は没。** 各技法は「操作手順」と「やり過ぎの寒さ」を対で守れ。

### 4-1. 志賀直哉 / カーヴァー(氷山理論・言い落とし)
- 手順:①心情語を一度書き、必ず消す。消した後に「その時、手 / 目 / 物が何をしていたか」を1つだけ書く。②因果を説明せず、2つの事実を接続詞なしで並置し判断を読者に渡す。③感情のピーク直前で「〜た。」で止める。
- 断定は手段であって様式ではない。**断定短文の連発が電文調になったら崩す。**
- 寒さ:事実の後に「＝つまり緊張していた」と種明かしすると氷山が崩れる。「手が震えていた」等、物を意味ありげにしすぎると結局は感情語の言い換え。

### 4-2. ルシア・ベルリン(自己憐憫ゼロの乾いた一滴・位置と長さ)
- 手順:①つらい場面はつらさを説明せず次の行動へカメラを回す。②一滴は感情の谷でなく「谷を通り過ぎた平地」に置く。③自己言及・自己弁護をしない。事実の並置で苦笑させる。④長さは最長二文。滴らせたら次の文は平常の事実に戻して即引く。
- 寒さ:一滴を延ばし「情けない話だが」と自己註釈を足すと湿って泣き言。痛みの直後に置いても泣き言。毎場面に滴らせると演技。

### 4-3. 向田邦子(生活の物・湿らせない処理)
- 手順:①場面に「この稽古場 / この相方に固有の生活の物」を1つ具体で置く。②その物を感傷で説明せず、所作(誰がどう扱ったか)だけ書く。「大事に」「寂しく」等の副詞を禁じる。③標準語一人称の常体で、語尾は自分を少し突き放す。④薬味は一皿に一つ。
- 寒さ:物に来歴を語らせると説明過多。副詞で湿らせた瞬間に演歌調。

### 4-4. しんせかい / 山下澄人(短文・句点・改行余白で"間")
- 手順:①一文を短く。接続助詞で連ねず句点で切る。②改行を沈黙として使う。重い事実の後に一行空け、次の文で説明を続けない。③主語・目的語を文脈で落とす。ただし体言止めの連打は避ける。
- 寒さ:改行と短文の機械的濫用はポエム臭。全文体言止めは余白が「スカスカ」。余白は重い一文の後に一度だけ効く。

### 4-5. 火花 / 又吉(下積みの非対称・世界の事実の積み方 ※声は移植しない)
- 手順:①非対称を「二つの事実の並置」で見せる(売れた同期と俺たちを感想なしで隣に置き、差は読者に測らせる)。②金・時間・場所を具体数値【仮】で刻む(「貧しい」でなく「客席42脚、埋まったのは9脚」)。③世界のルールを説明でなく所作で見せる。④自己憐憫の代わりに勘定へ逃がす。
- **★声は移植しない:**「僕」・関西の思弁・比喩の連射・自分語りの長文をしない。思弁(「笑いとは何か」)を一行でも入れるとregisterがズレて全体が借り物臭くなる。

## 5. スロット地図

本数は**天井であってノルマではない。** 浅い埋め草が最大の失敗。優先順の上位から彫る。反復回避は「直近使用ID方式」前提で、同一スロット内で既読感を薄める並びで書く。

**★内心は上から1つだけ発火する排他カスケード(低体力>金欠>連敗>直近通過>大会前>平常)。** 各スロットは"他の高優先が偽のとき"に出る前提で書く(金欠中に連敗台詞は出ない)。**谷口・相方の発話も§2-1の対象**(生活・段取り・観測・軽い事実のみ)。
