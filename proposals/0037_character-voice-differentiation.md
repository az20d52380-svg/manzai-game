# 第1部 役割語研究（キャラクター言語の理論）

# 日本語フィクションの「キャラクター言語」調査 — セリフだけで話者を識別させる技法

## 1. 理論的基盤：金水敏の「役割語」研究

**定義**：ある言葉遣い（語彙・語法・言い回し・イントネーション）を聞くと特定の人物像（年齢・性別・職業・階層・時代・性格等）が思い浮かぶ、あるいは人物像から言葉遣いが思い浮かぶとき、その言葉遣いを「役割語」と呼ぶ（金水敏、2000〜。主著『ヴァーチャル日本語 役割語の謎』岩波書店）。（[Wikipedia「役割語」](https://ja.wikipedia.org/wiki/%E5%BD%B9%E5%89%B2%E8%AA%9E) / [岩波書店](https://www.iwanami.co.jp/book/b625302.html) / [大阪大学 ResOU](https://resou.osaka-u.ac.jp/ja/story/2019/fyvba9)）

重要な性質：

- **現実には誰も使っていなくても機能する**。「わしは知っておるんじゃ」という〈老人語〉を話す首都圏の老人はほぼ実在しないが、読者は即座に老人像を想起する。役割語は「ヴァーチャル日本語」＝読者と作者が共有するステレオタイプ知識である。（[国際交流基金 日本語教育通信](https://www.jpf.go.jp/j/project/japanese/teach/tsushin/research/201302.html)）
- **役割語度**という尺度がある。標準語＝役割語度0（誰の言葉でもない中立）で、周縁キャラほど役割語度が高い。**主人公・ヒーローは読者が自己投影しやすいよう役割語度の低い標準語＋α（αが個性）で話し、脇役・出番の少ないキャラほど濃い役割語（田舎ことば等）を割り当てられる**という非対称の法則（『ヴァーチャル日本語』）。（[金水敏 講演資料PDF](https://www.rikuryo.or.jp/activity/tokyo_club/wp-content/uploads/2024/09/%E3%80%8C%E5%BD%B9%E5%89%B2%E8%AA%9E%E3%80%8D%E3%81%8C%E5%88%87%E3%82%8A%E9%96%8B%E3%81%8F%E3%80%81%E8%A8%80%E8%AA%9E%E3%81%AE%E5%9C%B0%E5%B9%B3%EF%BC%88%E9%87%91%E6%B0%B420240302%EF%BC%89.pdf) / [Wikipedia](https://ja.wikipedia.org/wiki/%E5%BD%B9%E5%89%B2%E8%AA%9E)）
- 歴史的起源も研究されている。〈博士語・老人語〉は江戸期上方語に、〈女ことば（〜てよ/〜だわ）〉は明治の女学生「てよだわ言葉」に遡り、現代の日常会話ではほぼ消滅しているのにフィクション・翻訳内で生き続けている。（[Kijidasu「てよだわ言葉」](https://kijidasu.com/?p=45729) / [川村学園紀要PDF](https://www.jstage.jst.go.jp/article/kotoba/46/0/46_3/_pdf/-char/ja) / [川村インターナショナル「女ことばは翻訳から生まれる？」](https://www.k-intl.co.jp/blog/B_220420A)）

## 2. なぜ「名前を隠しても誰か分かる」のか

1. **一人称の多重体系**：日本語は「私/わたくし/僕/俺/あたし/わし/おいら/拙者/自分/うち…」と一人称だけで年齢・性別・階層・時代・自己認識を同時に伝える。一人称の選択自体が「自分はこういうキャラだ」という役割の主張になる。（[日本語の一人称代名詞 Wikipedia](https://ja.wikipedia.org/wiki/%E6%97%A5%E6%9C%AC%E8%AA%9E%E3%81%AE%E4%B8%80%E4%BA%BA%E7%A7%B0%E4%BB%A3%E5%90%8D%E8%A9%9E) / [かたすみ帳 一人称一覧](https://katasumityo.com/first-person/)）
2. **文末の可変性**：日本語は文末（コピュラ・終助詞）に話者情報を載せる余地が大きい。金水はこれを「キャラ語尾」と命名し、定延利之がさらに**キャラコピュラ**（だ→じゃ／や／ですわ）、**キャラ助詞**・**キャラ終助詞**（ぜ/わ/のう/っす等）に分類した（定延『日本語社会 のぞきキャラくり』）。有名例：バカボンのパパ「〜なのだ」、ラム「〜だっちゃ」、FFのモーグリ「〜クポ」——命題内容と独立に話者だけを標示する。（[三省堂 定延コラム「キャラ助詞再論」](https://dictionary.sanseido-publ.co.jp/column/chara20) / [同「役割語について」](https://dictionary.sanseido-publ.co.jp/column/chara-hoi-99) / [金水 発表資料PDF](http://skinsui.cocolog-nifty.com/sklab/files/yakuwarigokaramitanihongotokyaranihongobunpogakkai.pdf)）
3. **ステレオタイプの経済性**：役割語を使えば人物説明ゼロで人物像を一発提示できる（だから翻訳でも現実に存在しない「女ことば」が多用される）。これが識別効果の源泉であり、同時に弊害の源泉でもある。（[川村インターナショナル](https://www.k-intl.co.jp/blog/B_220420A)）
4. **計量的裏付け**：小説のセリフは文末形式（〜だぜ/〜ですわ/〜なのだ等）の分布だけで話者を統計的に識別できることが工学系研究でも示されている（佐藤理史・名古屋大）。（[NINJAL LRW2024論文PDF](https://clrd.ninjal.ac.jp/lrw/lrw2024/o16-paper.pdf)）

## 3. 使いすぎの弊害と、上手い作品の回避法

**弊害**（研究側の指摘）：

- 役割語はステレオタイプそのものなので、安易な多用は**偏見・差別の再生産**（方言＝田舎者、女ことば＝従属的女性像などのジェンダーギャップ含む）につながる。（[大阪大学 Dialogue](https://dialogue.osaka-u.ac.jp/282/) / [小林美恵子「役割語における差別を考える」PDF](http://gendainihongo.sakura.ne.jp/gendainihongo/wp-content/uploads/2019/04/34-%E5%BD%B9%E5%89%B2%E8%AA%9E%E3%81%AB%E3%81%8A%E3%81%91%E3%82%8B%E3%80%8C%E5%B7%AE%E5%88%A5%E3%80%8D%E3%82%92%E8%80%83%E3%81%88%E3%82%8B.pdf)）
- 型通りの役割語の多用は**日本語本来の多様性を覆い隠し、陳腐さ・表現の狭さ**をもたらす。（[Wikipedia](https://ja.wikipedia.org/wiki/%E5%BD%B9%E5%89%B2%E8%AA%9E)）

**創作実務側の指摘**：語尾・一人称でのキャラ付けは「手っ取り早いが誰でもできる」ため、それだけだと**わざとらしく・寒く・薄い記号キャラ**になる。キャラ立ちの本体は中身（価値観・行動）であり、口調は補助線に過ぎない。（[note 石黒抱魔「語尾と一人称でキャラ付けするんじゃない」](https://note.com/croquettesofmars/n/n49ae8c819b9d) / [小説家になろう「語尾・人称によるセリフのキャラ付け考察」](https://ncode.syosetu.com/n4943du/) / [カクヨム「キャラクターの口調に関する問題への私見」](https://kakuyomu.jp/works/1177354054884365739/episodes/1177354054886243573)）

**上手い作品の回避手筋**：

- **役割語度の傾斜配分**：主要キャラは標準語＋微差（役割語度低）、濃い役割語は脇役・一発キャラに限定する（金水の「ヒーローは標準語」則の応用）。全員が濃い語尾だと世界が記号化する。
- **役割語の「混ぜ」で立体化**：金水が『もののけ姫』分析で挙げるジコ坊は、僧侶らしいいかめしい言い回しと俗語が混在し、その**ズレ自体が「謎めいた人物」という性格情報**になる。単一カテゴリの純粋培養ではなく、2系統のブレンドや場面による切替が人物に奥行きを与える。（[ほとんど0円大学 もののけ姫×言語学](https://hotozero.com/knowledge/osakauniv_linguistics_mononokehime/) / [『役割語とキャラクター』研究社](https://www.kenkyusha.co.jp/book/b10135168.html)）
- **相手・状況で口調を変える**：目上には敬語、相方にはタメ口、緊張時は短く途切れ、平常時は長く——「変化する口調」は記号でなく人格として読まれる。呼称（あんた/お前/〜さん/呼び捨て）は関係性まで同時に伝える。（[monokakilabo](https://monokakilabo.com/dialogue-sentence/) / [榎本メソッド](https://enomotomethod.jp/column/conversationalsentences/)）
- **役割語＋価値観・反応原理の二層構造**：同じ質問への「何に反応し、何を無視し、何を先に言うか」が語尾より強い識別子になる。語尾を消しても価値観で誰か分かる状態を作り、語尾は薄く残す。（[すずしろブログ](https://suzushiroblog.com/2020/12/07/serifu-kakiwake/) / [創作おTips](https://note.com/kashiri/n/nc7eebbee2390)）

## 4. 操作可能なレバー一覧（規約設計用）

| レバー | 選択肢の軸 | 効果・注意 |
|---|---|---|
| **一人称** | 俺/僕/私/わたくし/わし/うち/自分/名前呼び | 最強の識別子。1キャラ1固定が原則。ひらがな/カタカナ表記（僕/ぼく/ボク）でも印象が変わる |
| **二人称・呼称** | お前/あんた/君/〜さん/あだ名/呼び捨て | 関係性と上下感を同時に伝える。相手ごとに変えると人格が立つ |
| **語尾・終助詞** | ぜ/さ/な/のう/わ/かしら/っす/〜じゃん | 効くが最も陳腐化しやすい。頻度を落とす（毎文でなく数文に1回）のが実務上の定石 |
| **コピュラ（断定形）** | だ/です/じゃ/や/である/ですわ | 年齢・地域・階層を一語で標示。役割語の中核 |
| **敬語度** | 無敬語/タメ口/です・ます/謙譲過剰 | 「誰に対して変えるか/絶対に変えないか」が性格情報になる |
| **文の長さ・リズム** | 短文連打型/長文説明型/体言止め | 語尾に頼らない識別子。感情状態で伸縮させると生きる |
| **語彙レベル** | 漢語・専門語/平易語/俗語/流行語 | 知性・階層・世代を示す。漢字多め=硬質、ひらがな多め=幼く柔らかい |
| **言い淀み・フィラー** | えー/あのー/…/言い直し | 自信・誠実さ・動揺を示す。頻度差だけでキャラが割れる |
| **擬音・間投詞** | 感嘆の型（うおっ/まあ/ほう） | 反射的な音の癖は理性的なセリフより人格が漏れる |
| **口癖（内容語）** | 決まり文句・持ちネタ的言い回し | 語尾より自然な識別子だが、乱発すると同じく記号化する |
| **話題選択・反応原理** | 何に食いつくか/何をスルーするか/金・義理・見栄など判断基準 | 最深層のレバー。語尾を全部剥がしても識別できる状態の担保。役割語はこの上の化粧 |
| **口調の可変域** | 誰の前で崩れるか/本音モードの切替条件 | 「変わる瞬間」がドラマの山場を作る。固定口調キャラとの対比で使う |

**規約への示唆**：(1) キャラごとに上表のレバー値を仕様として固定する（一人称・呼称・コピュラ・敬語度は必須、キャラ語尾は任意かつ低頻度）。(2) 主人公級は役割語度を低く、脇役ほど高く傾斜させる。(3) 「語尾を機械的に置換しただけで成立するセリフ」を検収NGとし、価値観・反応で識別できることを一次条件にする。(4) 方言・性差系の濃い役割語は差別的ステレオタイプ再生産のリスクを規約に明記する。

# 第2部 脚本術の書き分け実務（cover the names・9レバー）

# キャラの声の書き分け：脚本術の実務技法（調査結果）

## 1. 「名前を隠しても誰か分かるか」テスト（cover the names）の出典と運用

- **定式化した出典として最も引用されるのは Scriptnotes（John August & Craig Mazin）Ep.37 “Let's talk about dialogue”（2012）**。ここで「キャラの声が機能しているかのテスト」群が提示され、No Film School が「Five Tests to See Whether a Character's Voice is Working」として整理した。テストは概ね次の5つ。
 1. **入れ替えテスト**：あるキャラの台詞を別のキャラに言わせても成立してしまうなら声が無い（成立するなら2人は統合候補ですらある）。
 2. **名前隠しテスト**：キャラ名を隠して（あるいは全員同じ名前にして）読んでも話者を特定できるか。
 3. **俳優想像テスト**：各キャラに特定の俳優の声を当てられるか。「この俳優は絶対この台詞を言わない」と除外できるなら声が立っている。
 4. **ジョークテスト**：そのキャラが冗談を言うとしたらどう言うか想像できるか（ユーモアの型はそのキャラの状況対応全般の縮図になる）。
 5. **作者声テスト**：全員が「書き手自身の声」で喋っていないか。
 出典: [Scriptnotes Ep.37](https://johnaugust.com/2012/dialogue) / [同トランスクリプト](https://johnaugust.com/2012/scriptnotes-ep-37-lets-talk-about-dialogue-transcript) / [No Film School まとめ](https://nofilmschool.com/2012/05/tests-characters-voice-working-from-john) / 続編 [Scriptnotes Ep.609 “Dialogue and Character Voice”](https://johnaugust.com/2023/scriptnotes-episode-609-dialogue-and-character-voice-transcript)
- **運用**は業界で広くルーチン化されている。[ScreenCraft「5 Simple Tests Your Screenplays Need to Pass」](https://screencraft.org/2018/10/25/5-simple-tests-screenplays-need-pass)は「全キャラを同じ名前（例: 全員LARRY）に置換して読み、声を判別できるか」という機械的な実施法を提示。[Screenplay Readers](https://screenplayreaders.com/screenplay-dialogue/)も「名前を覆っても話者が分かる状態」を到達目標として明記する。
- **落ちたときの修復手筋**（各記事の共通見解）：語彙とケイデンス（話す速さ・切り方）を出自・背景に合わせて調整する／構文（語順・文の組み立て）を変える／その人固有の言い淀み・言い直し（disfluency）を設計する。出典: [No Film School「Characters Who Sound Nothing Like You」](https://nofilmschool.com/dialogue-sounds-nothing-like-you)

**ゲームへの運用示唆**：テキスト規約の受け入れ基準として「話者名を伏せたレビューで、相方／谷口／俺の心の声／各審査員を判別できること」をゲート化できる。判別できない台詞は「どのキャラに統合してもよい台詞＝声が無い台詞」としてリジェクトする。

## 2. アンサンブルもので声を書き分けるレバー（欧米脚本術の整理）

- **価値観（worldview）→同じ状況への異なるリアクション**：台詞の個性は「何を言うか」より先に「その状況をどう解釈するか」で分岐させる。楽観主義者と皮肉屋は同じ事実に別の言葉を選ぶ。出典: [Greenlight Coverage](https://glcoverage.com/2024/06/14/how-to-give-characters-unique-voices-in-screenplay/) / [Industrial Scripts](https://industrialscripts.com/writing-voice/)
- **want/need とサブテキスト**：John Truby 系の整理では、want（表の欲求）と need（本人が自覚しない欠落）のギャップがサブテキストの摩擦になる。群像シーンでは「各キャラに隠れた欲求を持たせ、それらを互いに衝突させる」のが定石（Truby『The Anatomy of Story』のシーン構築・“symphonic dialogue”）。出典: [Truby要約（Medium）](https://medium.com/@pirangy/scene-construction-and-symphonic-dialogue-john-truby-the-anatomy-of-story-p-373-376-20c512cab169) / [Subtext論（Substack）](https://cmerritthoughton.substack.com/p/story-subtext-is-the-drawing-problem)
- **文の長さ・リズム（competing cadences）**：短く刻むキャラと長く回すキャラを同席させると、シーンに摩擦とテクスチャが生まれる。緊急性は短文、内省は長文で担う。出典: [Fiveable: Character voice and speech patterns](https://fiveable.me/writing-the-episodic-drama/unit-5/character-voice-speech-patterns/study-guide/3cBB4VwmceIRxwyw) / [Fiveable: Dramatic dialogue techniques](https://fiveable.me/tv-writing/unit-4/dramatic-dialogue-techniques/study-guide/9m29ZVivt2qSblkK)
- **語彙水準（register）**：難語／平易語、抽象／具象、業界用語の有無。事前に「キャラ別ボイス仕様書」（典型語彙・口癖・言い淀み・リズム）を1枚作ってから書くのが実務推奨。出典: [Fiveable: Principles of Effective Dialogue Writing](https://fiveable.me/screenwriting-i/unit-5/principles-effective-dialogue-writing/study-guide/G9n3btidIrlf5sHc)
- **構文（syntax）**：最も難しいが最も効くレバーとされる。語順・文構造そのものをキャラ化する（例として、常に整然と正確な文で話すキャラ、倒置で話すキャラ＝『The Curious Incident』のクリストファーやヨーダの「型」が挙げられる）。出典: [No Film School](https://nofilmschool.com/dialogue-sounds-nothing-like-you)
- **応答様式**：質問で返す／断定で返す／答えずに話題をずらす、の型を人物ごとに固定する。群像シーンでは全員に均等な発話量を与えず、「よく喋る役」と「リアクションで存在する役」に配分する。出典: [Fiveable: Dramatic dialogue techniques](https://fiveable.me/tv-writing/unit-4/dramatic-dialogue-techniques/study-guide/9m29ZVivt2qSblkK)
- **台詞以外の反応**：言葉と身体反応（行動・沈黙）の矛盾でキャラを分ける。「言うこと」と「すること」の距離自体がキャラ変数。出典: [Final Draft blog](https://www.finaldraft.com/blog/how-to-give-your-characters-unique-voices-in-your-screenplays)

## 3. 日本の会話劇の書き分け

- **三谷幸喜：当て書き**。先に演者を決め「この人がこれをやったら面白い」から台詞を書く。全台本が当て書きのため、演者が替われば台本を改訂するほど「声＝特定の身体」に結びつけている。群像劇では周辺人物のリアクションが物語を駆動し、笑いの場面でも各人が深刻な事情を背負う（＝同じ場面で各キャラの temperature が違う）ことを設計段階で整理する。出典: [Wikipedia 三谷幸喜](https://ja.wikipedia.org/wiki/%E4%B8%89%E8%B0%B7%E5%B9%B8%E5%96%9C) / [三谷本人の当て書き論（ありふれた生活1080）](https://plaza.rakuten.co.jp/foret/diary/202203310003/) / [文春オンライン](https://bunshun.jp/articles/-/46974) / 一次資料は『三谷幸喜 創作を語る』（[講談社](https://bookclub.kodansha.co.jp/product?item=0000396333)）
- **坂元裕二：「普通の人なら言わないこと」を言わせる**。シナリオ・センターの『最高の離婚』分析記事は、キャラが立つ台詞の作り方を「普通の人なら言わない言い方を探すほど個性が際立つ」と定式化（[シナリオ・センター](https://www.scenario.co.jp/online/21576/)）。本人は「結末を決めず、登場人物と一緒に生きながら書く」「役者が揃っているときは芝居の邪魔をしない（＝声を薄くする判断もある）」と語る。出典: [CREA インタビュー連載](https://crea.bunshun.jp/list/feature/crea201810-sakamotoyuji) / [同・カルテット回](https://crea.bunshun.jp/articles/-/21028) / [Yahoo!ニュース特集](https://news.yahoo.co.jp/feature/1093/)
- **宮藤官九郎：「ウソの数を減らす」**。固有名詞を台詞に入れ、劇中人物が観客と同じ日常会話をしているように見せる＝語彙の具体度・固有物への言及量そのものをキャラのリアリティ装置にする。出典: [ぴあ インタビュー](https://lp.p.pia.jp/article/news/460376/index.html) / [マイナビニュース](https://news.mynavi.jp/article/20240624-kudokan/)
- **シナリオ・センターの一般化**：性格を「〜すぎる」で規定（律儀すぎる・调子よすぎる等）すると、同じ内容の一言でも人物ごとに言い回しが自動的に変わる、という訓練法。「ありがとう」一つでも丁寧に言うかぶっきらぼうに言うかで人物が立ち上がる＝台詞は人物の「声」であり、受け手は声から人柄を読む。出典: [シナリオ・センター「いいセリフを書くには」](https://www.scenario.co.jp/online/21259/)
- **役割語（金水敏）**：一人称（私/僕/俺…）と文末表現（〜じゃ/〜だわ/〜ぜ…）だけで人物像を想起させる日本語特有の装置。創作用の整理として『〈役割語〉小辞典』がある。計量研究では、堅物型キャラは二人称・助動詞・文末助詞に、自由奔放型は感動詞・感情語に特徴が出るという分析もある。**ただし役割語はステレオタイプの近道なので、規約では「役割語に頼らず、下記の構造レバーで書き分け、役割語は最後の微調整」とするのが安全**。出典: [役割語 Wikipedia](https://ja.wikipedia.org/wiki/%E5%BD%B9%E5%89%B2%E8%AA%9E) / [大阪大学 ResOU](https://resou.osaka-u.ac.jp/ja/story/2019/fyvba9) / [国際交流基金](https://www.jpf.go.jp/j/project/japanese/teach/tsushin/research/201302.html) / [計量国語学34巻6号（アニメ台詞の計量分析）](https://www.jstage.jst.go.jp/article/mathling/34/6/34_389/_pdf)

## 4. 本題：同じ一言（例：「負けた」の報告）を4人に言わせたらどう変わるか — 書き分けレバー一覧

前提：**変えるのは「事実」ではなく「事実の通し方」**。同じ情報（負けた）を、以下のレバーの組み合わせで別人の声にする。既存作品の台詞は使わず、型のみ記す。

### レバー1：解釈フレーム（価値観）— 何のせいにするか
同じ敗北を「自分の実力」「相手の強さ」「運・環境」「システムの理不尽」のどれで語るかを人物ごとに固定する。これが最上流で、以下の全レバーの方向を決める。

### レバー2：want/need のズレ — 報告で本当は何が欲しいか
- 慰めが欲しいのに強がる（言葉と欲求が逆向き＝サブテキスト最大）
- 分析を聞いてほしい（敗因を先に喋り出す）
- 責任の所在を確定したい（主語が「俺」か「あいつ」かで割れる）
- 次の話がしたい（敗北の報告を最短で切り上げる）

### レバー3：情報の出し順 — 結論から言うか、外堀から埋めるか
- 断定型：一語で結論→補足なし
- 回避型：状況説明を長く前置きして、肝心の一語を最後まで言わない（または言わずに伝わらせる）
- 分析型:結論→即、数値や敗因の列挙へ
- 自罰型：結論より先に謝罪・自己評価が出る

### レバー4：発話の長さとリズム
同じ報告を「一語」「一文」「三文」「言い直しを含む長回し」に配分。短いキャラは常に短く、長いキャラは常に長く（competing cadences）。沈黙・言い淀み（disfluency）も長さの一種として設計する。

### レバー5：応答様式 — 質問か断定か
報告後の第二声で分ける。「で、どうする？」と未来の質問で返すキャラ／「原因はこれだ」と断定するキャラ／「そっちはどうだった？」と話を外すキャラ／何も足さないキャラ。

### レバー6：語彙水準と具体度
- 抽象語で語る（「実力不足」型）か、固有物・数字で語る（点数・会場名・ネタ順など具体に逃がす型＝宮藤の「固有名詞」原理、かつ本プロジェクトの「感情語を書かず固有物に背負わせる」規約と同型）
- 敬体/常体、業界用語の使用量もここ

### レバー7：構文
倒置の有無、体言止めの頻度、主語の省略癖、「〜けど」で文を終える癖など、文の骨格自体を人物に割り当てる（最難だが最も判別性が高い）。

### レバー8：言葉と身体の矛盾（ト書き側レバー）
「平気だ」と言いながら物をしまう手が止まる、等。台詞を同じにしても行動指示で別人化できる。台詞テキストだけのゲームでは「心の声と発話のズレ」で代替可能。

### レバー9：役割語（最後の微調整）
一人称・文末助詞・感動詞。効果は強いが記号的になりやすいので、レバー1〜8で判別可能にした後の仕上げに限定する。

### 運用ゲート（規約への落とし込み案）
1. キャラごとに「解釈フレーム／want-need／出し順／長さ／応答様式／語彙水準」の6行ボイス仕様を先に固定する。
2. 書いた後、話者名を伏せて並べ替えた「負けた報告」4連を第三者（または別セッション）が判別できるかテスト（cover the names の運用）。
3. 判別不能な台詞は、まずレバー1（解釈フレーム）から直す。語尾（レバー9）だけで直すのは禁じ手（記号化するため）。

# 第3部 口癖・決め台詞の三類型と運用

# 口癖・決め台詞の類型と運用ルール（アニメ・ドラマ・漫画調査）

※以下は構造・技法の抽出のみ。例示は作品名・キャラ名・型の説明に留める（セリフ本文の転載はしない）。

## 1. 三類型と頻度運用の違い

### 類型A: 日常頻出型（地の口癖）— 「高頻度でこそ成立」
- **中身**: 語尾型（『NARUTO』ナルトの「〜だってばよ」、『ケロロ軍曹』ケロロの「〜であります」、『うる星やつら』ラムの「〜だっちゃ」、『侵略！イカ娘』の「〜ゲソ」等）、相槌・接頭型、独特の一人称・呼称。キャラ語尾は「受け手にキャラを理解してもらう・特徴をつける」ための装置と整理されている（[ニコニコ大百科「語尾の一覧」](https://dic.nicovideo.jp/a/%E8%AA%9E%E5%B0%BE%E3%81%AE%E4%B8%80%E8%A6%A7)、[gooランキング「語尾で誰か分かるキャラ」](https://ranking.goo.ne.jp/column/5828/)、[マンガペディア](https://mangapedia.com/gd/yposal454)）。
- **運用ルール**:
  - 頻度: **ほぼ毎発話〜毎シーン**。「声を消しても誰の台詞か分かる」状態が目標。減らすと逆にブレ・キャラ崩壊に見える。
  - 置き場所: 平常時の会話全般。**シリアスの山場ではむしろ外す／薄める**のが定石（後述の類型Cと接続する）。
  - 注意: 語尾だけの記号的キャラ付けは単調化する。「言葉の選び方・話の組み立て方」にも個性を分散させよとされる（[小説作法「キャラの口調」](https://www.raitonoveru.jp/howto/h/271a.html)、[語尾・人称によるキャラ付け考察](https://ncode.syosetu.com/n4943du/)、[口調のデザイン法](https://urakatahero.hateblo.jp/entry/kuchou-omosirosa)）。

### 類型B: 決め台詞型（様式美・儀式）— 「頻度は低〜中、発動場面が固定」
- **中身**: 変身口上・名乗り・犯人指摘・登場/退場の口上。「決める時に使うからこその決め台詞」で、敵を倒す時・真相を明かす時・登場時・退場時など**発動条件がフォーマット化**されている（[ピクシブ百科事典「決め台詞」](https://dic.pixiv.net/a/%E6%B1%BA%E3%82%81%E5%8F%B0%E8%A9%9E)、[「お約束」](https://dic.pixiv.net/a/%E3%81%8A%E7%B4%84%E6%9D%9F)、[「様式美」](https://dic.pixiv.net/a/%E6%A7%98%E5%BC%8F%E7%BE%8E)）。
  - 例: 『金田一少年の事件簿』の「じっちゃんの名にかけて」型＝**事件解決を決意する局面限定**で発する誓いの口上（[ORICON 誕生秘話](https://www.oricon.co.jp/news/2344106/full/)、[ピクシブ百科事典](https://dic.pixiv.net/a/%E3%81%98%E3%81%A3%E3%81%A1%E3%82%83%E3%82%93%E3%81%AE%E5%90%8D%E3%81%AB%E3%81%8B%E3%81%91%E3%81%A6)）。『セーラームーン』の名乗り口上＋決めポーズ、仮面ライダー・プリキュアの変身口上は「変身→名乗りでワンセット」の儀式（[RENOTE](https://renote.net/articles/324128)、[プリキュア変身口上まとめ](https://www.hobiwo.com/entry/precurechange)、[仮面ライダー決め台詞一覧](https://maruburo.com/kamen-riders-signature-phrase/)）。
- **運用ルール**:
  - 頻度: **1エピソード（1事件・1戦闘）につき最大1回**。海外の脚本論でも「TVは1話1回、書籍なら1回、映画は1幕1回」「3話以上にまたがる3回の反復で観客に定着する」という目安が示される（[Writing Catchphrases – Marcus Goh](https://marcusgohmarcusgoh.com/point-of-entry/writing-catchphrases)）。
  - 置き場所: **決まったスロット（変身時・解決宣言時・とどめ時）以外では言わせない**。スロット固定こそが様式美の正体で、視聴者は「ここで来る」と予測して待つ（[ピクシブ百科事典「お約束」](https://dic.pixiv.net/a/%E3%81%8A%E7%B4%84%E6%9D%9F)）。
  - 効果: 主人公の信念・価値観を一発で記憶させ、子どもの模倣を誘う（[あにこれ 決め台詞タグ](https://www.anikore.jp/tag/%E6%B1%BA%E3%82%81%E5%8F%B0%E8%A9%9E/)）。

### 類型C: 感情の切り札型 — 「作品全体で数回」
- **中身**: 普段そのキャラが絶対に言わない種類の一言。普段コミカルな『銀魂』坂田銀時がシリアス局面で生き様を語る型＝「いつもは迷言ばかりだからこそ効く」ギャップとして読者に評価される。逆に「いつもニコニコのキャラが淡々と本音を言うと怖い」のも同型（[レビューン等の名台詞評](https://reviewne.jp/contents/197/quotes)、[アニメ！アニメ！名台詞ランキング](https://animeanime.jp/article/2021/05/18/61391.html)）。
- **運用ルール**:
  - 頻度: **1章〜1シリーズに1回級**。反復させない。2回目から「切り札」でなくなる。
  - 置き場所: 物語の転換点・最終盤。海外脚本論でも「第3幕の要所で使うと、それまでの感情が一気に回収され緊張が最大化する」とされる（[Marcus Goh](https://marcusgohmarcusgoh.com/point-of-entry/writing-catchphrases)）。
  - 前提条件: **類型Aの平常運転が十分に積まれていること**。「普段」が確立していないとギャップが発生しない。

## 2. 「乱発で死ぬ」のはどれか

- **死ぬのは B と C**。決め台詞はスロット外で使うと様式美が崩れて安売りになり、切り札は2回で切り札でなくなる。海外の実例批評では『グレート・ギャツビー』のギャツビーが台詞の約1/3で同じ呼びかけ（"old sport"型）を繰り返す点が「クライマックスでの反復は切迫感を出すが、それ以外の場所での乱発は正当化されない」と分析されている。「よく置けば絶大、乱発すれば興ざめ」（[Being a Better Writer: Catchphrases](https://maxonwriting.com/2023/11/06/being-a-better-writer-catchphrases/)）。また「全キャラに決め台詞を義務付けて詰め込むな。持つキャラと持たないキャラがいてよい」とも（同記事）。
- **頻出してこそキャラになるのは A**。ナルトの語尾、ケロロの軍隊口調は毎回出るから「地」になる。ただしAでも、日本の小説作法側は「口癖は差し色。ここぞで効かせ、『この言い回し＝このキャラ』と刷り込む程度に」と、**文章媒体では頻度をアニメより大きく下げる**ことを推奨している（[小説作法](https://www.raitonoveru.jp/howto/h/271a.html)、[創作に使える口癖一覧](https://sousaku-neta.com/favorite-phrase/)）。媒体差: 音声つきアニメは毎発話でも流せるが、文字媒体では同フレーズ反復が視覚的に目立ちノイズ化しやすい——**ゲームのテキストウィンドウは文字媒体側**に近い点に注意。

## 3. 口癖以外の反復識別子

| 識別子 | 型 | 例 | 運用 |
|---|---|---|---|
| 決めポーズ | 類型B相当 | セーラームーンの名乗りポーズ（口上とセットで儀式化）（[RENOTE](https://renote.net/articles/324128)） | 決め台詞と同じスロット固定・1場面1回 |
| リアクションの型（ランニングギャグ） | 類型A寄り | 「特定の刺激→毎回同じ過剰反応」の反復。クラウン芸の理論では**1公演に3〜4回**挟み、観客は「笑うタイミングが分かる」ことで演者に好感を持つ。ただし**最後の1回は予想を裏切って外す（ひねる）**のが飽き防止の定石（[ロネとジージ ランニングギャグ講座](https://roneandgigi.com/%E3%80%90web%E8%AC%9B%E5%BA%A7%E3%80%91%E3%83%A9%E3%83%B3%E3%83%8B%E3%83%B3%E3%82%B0%E3%82%AE%E3%83%A3%E3%82%B0%E3%82%92%E3%81%A4%E3%81%8F%E3%81%A3%E3%81%A6%E3%81%BF%E3%82%88%E3%81%86/8243/)、[ナムウィキ「ランニングギャグ」](https://ja.namu.wiki/w/%EB%9F%AC%EB%8B%9D%20%EA%B0%9C%EA%B7%B8)） | 中頻度・3回目で変化球 |
| 特定の話題への執着 | 類型A（話題トリガー型） | 『ONE PIECE』ナミ＝金銭が絡むと目の色が変わる（好物: 金・宝・みかん）（[Wikipedia ナミ](https://ja.wikipedia.org/wiki/%E3%83%8A%E3%83%9F_(ONE_PIECE))）。食いしん坊・規則マニア等も同構造 | 「トリガー話題が出たら必ず反応する」一貫性が本体。台詞の文言は毎回変えてよい＝乱発耐性が高い |

## 4. ゲーム規約向けの統合運用ルール（要約）

1. **1キャラの識別子は最大2系統**（例: 語尾＋執着話題）。全キャラに決め台詞を配らない。
2. **類型A（語尾・相槌・執着反応）**: 平常シーンで常時。ただしテキスト媒体では毎行でなく「差し色」濃度に落とし、文言バリエーションで回す。シリアス山場では薄める。
3. **類型B（決め台詞・決めポーズ）**: 発動スロットを仕様として固定（大会前口上・合格時など）。**1イベント1回・スロット外使用禁止**。定着には「3回以上の別エピソードでの反復」が必要なので、初出から一貫させる。
4. **類型C（切り札の一言）**: 1年（1シリーズ）に1回級の山場専用。事前に類型Aで「普段」を積んでから。使用後は当分封印。
5. **ランニングギャグ**: 同一リアクション3回目で必ずひねる（外す・裏返す）。
