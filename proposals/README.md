# proposals/ — 夜間リサーチの提案置き場（コード改変なし）

このディレクトリは **提案ドキュメント専用**。本体コードの変更差分は置かない。
オーナーが朝に中身を見て、良いものだけをオーナー or MVP側CLIが本体へ取り込む。

- 1提案＝1ファイル：`NNNN_<短い題>.md`
- 各ファイルの構成：**狙い / 対象箇所 / どこから学んだか(URL) / 実装イメージ(貼れるスニペット) / リスク・注意 / レッドチーム(2軸)**
- 会話・イベントの弱い/薄い/不自然な/伸ばせる箇所が中心。トーン・進行を壊さない範囲。
- **絶対の一線**：ゲームは"ボケ(オチ/ネタ本文)"を書かない。筆致にウィットは残す（過小申告・温度差実況等）。固有名は架空・数値は【仮】。
- 既存作品のセリフのコピペ禁止。学ぶのは"構造・間・分岐・笑いの型"。

索引は `../NIGHT_LOG.md` にサイクルごと追記。
