# character-voices — 人物別の声の識別子（書き分けの正本）

> これは種・数値は全て【仮】・最終確定は実機目視。
> オーナー裁定（2026-07-06）: **セリフは「レア在庫」でなく「キャラの識別子」**。アニメ・映画・ドラマのキャラ書き分け技法を調査し、人物ごとに分けたセリフを書くための正本がこのファイル。

## §1 原理 — 名前を隠しても誰か分かるか

### 出荷基準: 名前隠しテスト（cover the names）

脚本術の定番テスト（Scriptnotes Ep.37 ほか）: **話者名を伏せて読んでも誰のセリフか分かるか**。分からないセリフは「誰に言わせてもいい台詞＝声がない台詞」であり、没。入れ替えテスト（別のキャラに言わせても成立するか）も同じ判定に使う。

### 書き分けのレバー（優先順・上が本体、下が化粧）

1. **価値観・解釈フレーム**: 同じ事実（例: 敗退）を何のせいにするか・何に食いつき何をスルーするか。最上流のレバーで、これが下の全てを決める。
2. **その発話で本当は何が欲しいか**: 慰めが欲しいのに強がる／分析を聞いてほしい／次の話がしたい——言葉と欲求のズレがサブテキストになる。
3. **情報の出し順**: 結論一語で終える型／外堀から埋めて肝心を言わない型／結論→即敗因列挙の型。
4. **文の長さ・リズム**: 短文連打か長回しか。沈黙・言い淀み「……」も長さの一種。短いキャラは常に短く。
5. **応答様式**: 質問で返す／断定で返す／話をずらす／何も足さない。
6. **語彙水準・具体度**: 抽象語で語るか、固有物・数字で語るか。敬語度もここ。
7. **構文の癖**: 体言止め・同語反復・「〜けど」終わり・自分の宣言を自分で畳む、等。
8. **役割語（一人称・語尾・コピュラ）**: 最強の識別子だが最も記号化しやすい。**1〜7で判別可能にした後の仕上げに限定**。
9. **役割語度の傾斜**（金水敏の役割語研究）: 主人公は役割語度の低い標準語＋α、脇役ほど濃くてよい。**俺の識別子は語尾でなく価値観（計算屋・差引・段取り）**——だから俺に語尾の癖を作らない。

**禁じ手**: 判別できないセリフを**語尾の付け替えだけで直す**こと（記号キャラ化＝薄くて寒い）。直すときは必ずレバー1（解釈フレーム）から。

### 口癖の三類型（頻度は希少性でなく類型で決まる）

| 類型 | 中身 | 頻度・置き場所 |
|---|---|---|
| **A 日常型**（地の口癖） | 語尾・相槌・号令・執着話題への反応 | 頻出してこそ「地」になる。ただし**ゲームのテキスト窓は文字媒体**＝毎行でなく「差し色」濃度に落とし、文言バリエーションで回す。シリアスの山場では薄める |
| **B 決め台詞型**（様式美） | 発動場面が固定された口上・誓い | **発動スロット固定・1イベント1回**。スロット外で言わせると様式美が安売りになる |
| **C 切り札型**（感情の一言） | 普段絶対に言わないからこそ効く一言 | **年1〜2回級・感情の山専用**。2回目から切り札でなくなる。前提: A型の「普段」が積まれていること |

- 1キャラの識別子は**最大2系統**（例: 語尾＋執着話題）。全キャラに決め台詞を配らない。
- ランニングギャグ（同じ刺激→同じ反応の反復）は**3回目で必ずひねる**（外す・裏返す）。
- 執着話題型（金・規則・客席温度等）は文言を毎回変えられるので乱発耐性が高い＝増産の主力に向く。

## §3 話者判別ゲート（V表）— セリフを書いたら必ず走らせる

| # | チェック | ×のときの手筋 |
|---|---|---|
| V1 | **名前隠しテスト**: 話者名を伏せて読んで、この人物だと分かる（§2の表の「決め手」が読める） | レバー1（解釈フレーム）から直す——その人物なら**この状況を何のせいにし、何に食いつくか**を書き直す。**語尾の付け替えだけで直すのは禁止** |
| V2 | **入れ替えテスト**: 同じセリフを別の相方に言わせたら成立しない | 成立してしまうなら声がない＝§2の価値観・リズム・応答様式を注入して書き直す |
| V3 | 口癖の類型適合: A日常型は差し色濃度で自由／B決め台詞型は発動スロット内で1回／C切り札型（谷口「腹減ったな」等）は感情の山専用・年1〜2回 | 類型を§2で確認し、スロット外のB/Cを削る。Aの過密は文言バリエーションに散らす |
| V4 | 反応が「型」から出ている: 敗退・通過・金欠などの状況に対し、§2の「敗退時の反応型」と同じ解釈フレームで反応している | その人物の型（例: 早乙女=リテイク注文／立花=客席温度で再採点）に沿って発想し直す |
| V5 | 識別子の盛りすぎがない: 1セリフに識別子2系統まで。語尾・口癖・比喩・執着話題を全部載せしていない | 一番その人物らしい1系統を残して剥がす（記号の全部載せは薄くて寒い） |

**運用**: 相方・NPC・ライバルのセリフを含むバッチでは、A表・B表に加えてこのV表を走らせ、結果を出力する。複数人が同じ場面に出るときは「同じ一言を全員に言わせたら全員違う言い方になるか」を頭の中で1回回す（§2の敗退時の反応型の列がその見本）。
