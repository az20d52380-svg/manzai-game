## §1 芯（絶対の一線・5条）

1. **ネタ本文を一文字も書かない。** 笑いは地の文（心の声/場面描写/審査講評）にのみ宿る。ゲームが笑いを取りにいった瞬間に寒くなる。
   - 機械化チェック: 「この一行は、ネタのボケ・フリ・前提・オチ・パンチラインのいずれかを一文字でも本文化していないか？（＝観客に見せたら笑いを取りにいく台詞になっていないか）→一つでも該当すれば没。」
   - 谷口・相方の発話も対象。相方は生活・段取り・観測・軽い事実だけ喋る。
   - 例外: **審査講評が「オチ/ボケ/テンポ」を批評語として言及する**のは可（それ自体はネタ本文ではない）。
2. **固有名は全て架空。数値は全て【仮】。**
3. **既存作品（火花/べしゃり暮らし/M-1実在ネタ等）のセリフのコピペ・酷似は厳禁**（registerが汚染される）。火花からは世界の事実の積み方だけ学び、「僕」・関西の思弁・比喩の連射は移植しない。思弁（「笑いとは何か」）を一行でも入れると全体が借り物臭くなる。
4. **会話は話者ごとに1セリフずつ送る**（「／」で詰めない）。詰めると本番のタップ送りの"間"が死ぬ。データも1エントリ=1話者の1セリフ。
5. **方言ロック**: 谷口=関西弁で一貫／谷口以外の相方=標準語（例外は設定の芯が関西の二階堂のみ）／**俺の地の文・一人称は常に標準語**（「言うた」→「言った」）。第三者（店長・先輩・マネージャー等）にも関西弁を移さない（支配人・夜行列車・苦労人先輩など関西弁が設定の芯にある人物を除く）。

（出典: fable_kickoff_drama_allin_v1 §2、NIGHT_LOG.md:49、partner_finals_reactions_v0）

## §2 基調 — 感情を物に背負わせる

### 変換レシピ（1本ごとに必ずこの順）

1. 感情語を一度書く（下書き）。
2. 消す。消した跡に「その瞬間、手/目/物が何をしていたか」を**1つだけ**書く。
3. 因果を説明せず、2つの事実を接続詞なしで並置する。判断は読者に渡す。
4. 感情のピーク直前で「〜た。」で止める。

### やり過ぎの寒さ（レシピの裏面）

- 事実の後に「＝つまり緊張していた」と**種明かし**すると氷山が崩れる。
- 「手が震えていた」等、物を**意味ありげにしすぎる**と結局は感情語の言い換え。
- 「大事に」「寂しく」等の**副詞で湿らせる**と演歌調。物は所作（誰がどう扱ったか）だけで書く。
- 断定短文の**連発が電文調**になったら一箇所崩す。改行と短文の機械的濫用は**ポエム臭**。余白は重い一文の後に一度だけ効く。

### 固有/定番の原理（物の選び方）

- **固有** = その稽古場/相方/客席の固有名詞・履歴に紐づき、**他コンビの物語では成立しない物**。例: 相方が予約表に書く時の癖字、客席42脚のうち割れた9番、前のコンビが貼ったガムテの跡。
- **定番** = どのコンビにも置換可能な物。**置換可能なら没**。ブラックリスト: 冷めたコーヒー・蛍光灯・小銭・財布・家賃・天井の染み・雨・タバコの煙。
- **置換テスト**: その物を「別のコンビの物語」に移植して成立するか？ 成立するなら定番＝没。
- 物は**1テキストに1つだけ**。2つ目を足したくなったら、履歴の濃い方を残して消す。

（出典: fable_kickoff_drama_allin_v1 §3-A/§4、0009 §アンチ寒さ3則）

## §3 ユーモア規約 — 乾いた一滴

- **二本立てのみ**: (a) **自嘲** = 自己憐憫に転ばない乾いた自己言及（例の温度: 「うまくいかない時期が長いと、ネタより先に、頭の下げ方がうまくなる。」）／(b) **デッドパン** = 真顔で事実を並置・自己言及ゼロ・そのズレから苦笑が漏れる（例の温度: 「客は六人。そのうち一人は、開演五分で寝た。起きたとき、俺たちはもう終わっていた。」）
- 自嘲だけに絞ると湿った泣き言モノトーン、デッドパンだけだと達観キャラの演技。バッチ内で偏らせない。
- **頻度**: 分母は「プレイヤーが実際に連続で引く並び」。5〜6本に1本以下。1バッチ（同一スロットの一括執筆）最大1本。イベントは各分岐最大1個。これは**天井でありノルマではない**——機械的に6本目ごとに置くと予定調和のリズム＝別種の寒さになる。
- **位置**: 感情の谷でなく「谷を通り過ぎた平地」に置く。痛みの直後は泣き言に読める。
- **長さ**: 最長二文。滴らせたら次の文は平常の事実に戻して即引く。「情けない話だが」等の自己註釈を足すと湿る。
- 残りの4〜5本は**無味のドラマ地の文が正解**。平板＝手抜きではなく分母の設計。

（出典: fable_kickoff_drama_allin_v1 §3-A/§4-2、0009）

## §4 声のロック（人物別・やる/やらない）

| 声 | やる | やらない |
|---|---|---|
| **俺**（標準語・計算屋・ツッコミ体質・心配性） | 差引を数える。段取りを言う。観測する。感情を確率・勘定に逃がす | **キザ**（うまいこと言う警句・教訓・アフォリズムで締める）／**自己憐憫**／熱・執念・誇りの吐露／方言 |
| **谷口**（関西弁・楽観・大食い・実は繊細） | 生活・段取り・観測・軽い事実だけ喋る。本音は言語化せず「事実1つ」で置く（壁を名前で呼ぶ・三行目だけ二回読み返す） | 笑いを取りにいく台詞（フリ/ボケ/天丼のボケ側）／本音の吐露／標準語化 |
| **谷口以外の相方**（標準語・各キャラの芯は slots.md §5） | 各自の口癖・声プロファイルを厳守 | 関西弁（二階堂を除く）／谷口のモチーフの借用（「プロとして寝ろ」等） |
| **支配人**（関西弁・敬体混じり） | 年次の重さの事件化・システムの帯を人の声にする（「今年ぶんの伸びは使い切ったな」） | 出しすぎ（年1回・GP初戦前に一言【仮】が目安） |
| **第三者NPC**（標準語） | 実務の一言（slots.md §5の人物帳の声を厳守） | 関西弁の伝染／説明台詞 |
| **審査員7名** | 各人格の口調・型で「直せる一点」を含む講評 | 人格・才能・容姿の否定／実在審査員の想起 |
| **中立ナレーター**（三人称・標準語） | 「優勝は——〈　〉。」等の事実行 | 感想・演出過多 |

**熱の帰属ルール**: 熱・執念・壁の事件化は「俺」でなく谷口・支配人など**他者声へ寄せる**。俺に決意を書きたくなったら**段取りに変換する**（「明日、いちばんに来る。それでいい」）。俺の計算屋の目は差引を数えるためで、切れ味を見せるためではない。

**当事者性ルール**: 通過・敗退などコンビの結果は相方も当事者。**他人の祝辞・観客の感想の距離で喋らせない**（×「おめでとうございます」単体 → ○「通りましたね」）。例外は人物設定で担保できる場合のみ（楽屋挨拶文化等が人物帳に書いてある場合）。

**正典間の優先順位**: この表の一般規約より、**人物別の声プロファイル（slots.md §5・judge-personas.md=`../../manzai-judge-comments/references/judge-personas.md`）が優先**する。審査員の方言・比喩・小噺（花園の関西弁、神楽坂の比喩型等）は judge-personas.md の人格指定に従う——§1-5の「関西弁を移さない」は地の文・相方・NPCの規定であって、人格として関西弁が確定している審査員には適用しない。

（出典: fable_kickoff_drama_allin_v1 §3-B、dialogue_design_v0、partner_characters_v0、relationship_champions_v0:34）

## §5 技法は検算用（書いた後に見る・技法を成立させるために書かない）

主はドラマ（人物の事件と、感情を物へ転写すること）。技法名が先に立った行は没。書いた後、以下の寒さ側だけ確認する:

| 検算 | 寒さの出方 |
|---|---|
| 言い落とし（氷山） | 種明かし・意味ありげな物 → 感情語の言い換え |
| 一滴の位置と長さ | 谷の直後・二文超・自己註釈 → 泣き言／毎場面 → 演技 |
| 生活の物の処理 | 来歴を語らせる → 説明過多／副詞で湿らせる → 演歌調 |
| 間（句点・改行・余白） | 機械的濫用 → ポエム臭／体言止め連打 → スカスカ |
| 非対称の並置（売れた同期と俺たち等） | 差を口で言い切る → 警句化／思弁を挟む → 借り物臭 |

## §6 判定表（迷いを消す・断定）

| 迷い | 断定 |
|---|---|
| 感情を書きたい | 書かない。その瞬間の手/目/物を1つ書く（§2レシピ） |
| この物でいいか | 置換テスト。他コンビで成立するなら没。物は1個 |
| ユーモアを入れたいか | 先に分母を数える。並び5〜6本に1本を超えるなら入れない |
| 自嘲かデッドパンか | 自己言及があるなら自嘲、ゼロならデッドパン。それ以外の笑い（地口・誇張・擬人化の連発）は入れない |
| 俺に熱い決意を言わせたい | 言わせない。段取りに変換するか、熱は谷口/支配人のセリフへ移す |
| うまい一文が書けてしまった | 疑え。警句・アフォリズムなら消して、平常の事実で終える |
| 谷口の気持ちを書きたい | 言語化しない。所作か段取りの発話を事実1つで置く |
| 方言はどうする | 谷口・二階堂・支配人・夜行列車・苦労人先輩=関西弁。それ以外は全員標準語。地の文は常に標準語。**審査員だけは judge-personas.md の人格指定が優先**（花園=関西弁） |
| 壁（連敗・停滞）をどう名指す | 抽象的な連敗数でなく**回戦名＋経過年**（「また二回戦。三年、同じところにいる」）。**ただし回戦が状態から確定する文脈（ラダー会話・フック・テンプレ補間）限定**——静的プール（losing等）に回戦名を書き込まない（slots.md §1） |
| つらい場面をどう終える | つらさを説明せず、次の行動・平常の事実へカメラを回す |
| 敗退直後に励ましたい | 励まさない（押し売りは感動ポルノの縮小版）。嘆きもしない。事実で止める |
| 選択肢イベントで選択内容に触れたい | 選択肢の音読をしない（「締めるか、乗るか」はメニュー臭）。二択はボタンに委ねる |

## §7 尺の掟

毎週出る面（平常の心の声/予選講評/稽古地の文）= **短文・タップ送りの間だけで成立・冒頭で掴む**。地の文は1画面2〜4行、会話は話者ごと1セリフで別送し1画面に2話者を積まない。
長尺解禁は**E軸のみ**: 決勝敗退・優勝（勇退エンド）・年次リザルト独白。

## §8 天丼レジストリ規約（口癖・小物の再登場）

- 口癖は希釈すると死ぬ。**年2回まで・うち1回は感情最大点**（谷口「腹減ったな」は決勝進出/優勝級に温存）。この年2回制は谷口専用でなく**全相方の登録口癖**（slots.md §5の口癖列）に適用する。
- **反復発火するスロット（イベント分岐・最頻プール）に登録口癖を置かない**——1回書いた口癖が毎年複数回発火して予算を食い潰す。どうしても置くなら頻度管理（GameConfig側ローテ）をオーナーに相談してから。
- 天丼素材の近縁変種（「いけるわ」「いける気がすんねん」等）を最頻プールに混ぜない（隔離方針）。
- 3回目が最大。再登場は状況を変えて回す。最短再登場間隔はGameConfig管理【仮】＝テキスト側で勝手に決めない。

## §9 恒久規約

- **「笑」の字不使用**（オーナー方針・SNS想起回避）。代替: ウケる/沸く/客席が揺れる。例外域（据置）: 諺・慣用句・情緒的な「笑う」・「笑顔」。
- **実在審査員・実在芸人の模倣NG**。自己チェック: 「見た人が特定の実在人物*その人*を思い浮かべるか？」→浮かべばNG。
- **説明台詞禁止・数値の復唱禁止**（UIが言うことを人間に言わせない。「メンタル+1」と喋らせない。システム名を口にしない）。
- 性別を勝敗・煽りの装置にしない。
- 禁止2語（決勝敗北まわり）: 相方が優勝済ライバルに対して「先を越された/先に行かれた」を使わない（未優勝セルのみ可）。

## §10 摩耗ホットスポットと平板8割

- 最速で飽きるセル（最深プールが要る）: **決勝B帯講評・平常の心の声・晩年の講評**。
- **見せ場は全体の2割**。残り8割は「今週はネタを二本、頭から通した。まだ声には出していない。」級の平板が正解。毎行を山にした瞬間に寒くなる。
- 浅い量産が最大の失敗。本数は天井でありノルマではない。寒い/キザ/自己憐憫が出たら即引く——延ばして救おうとせず、物を1個に減らし、感情語/警句を消し、次の文を平常の事実に戻す。
