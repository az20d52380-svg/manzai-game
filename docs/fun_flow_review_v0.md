## 1. 中だるみの所在: 4〜7年目の「停滞レジーム」（1年単位でなく状態として発生）

- 初回プレイの主役は実測で停滞層である（0pt・裏天井ON時点で 分散63%/のんびり74% が9年目まで決勝未到達）。この層の4〜7年目は悪条件が重なる: **①カレンダー暗記済み**（3年目で完了・red_team指摘・年替わりフレーバーは🎯のまま）②**成長予算の年間増分が逓減**（6.0→年4で4.8→年7で3.6）＝週送りの差分ポップが年々痩せる ③**GP序盤3回戦（ライン18/26/34）が実力60前後には全て消化試合**＝GP季の6イベント中、緊張があるのは準々と準決だけ ④雑談・レポ在庫が6年目以降枯渇（red_team・増産中）。裏天井（8年目〜）と最終年（10年目）が晩年を救う設計になった今、**体験の谷は構造的に4〜7年目に固定される**。
- 週送りの手触り自体（3秒ルール・差分ポップ・伸びしろゲージ）は正しく設計済み。問題は「良い週送りで送る中身」が4〜7年目に薄いこと。

## 2. 感情アークの最弱点: 「同じ壁への連敗」に物語の階段がない

- 決勝の高揚・敗北の悔しさ・才能解放の火・ラストイヤー・裏天井の景色——山は全て設計済み。**最弱はその手前の反復敗退**: 停滞層の典型は準決（74）or 準々（45）の同じ壁に3〜5年連続で跳ね返されるが、1年目の「地図を作る年」（支配人）以降、**壁との再会を語る言葉が10年目までない**。飽きられデバフ等の機構は既定OFFで正しい（数値で語る必要はない）が、テキストも沈黙しているため、最も回数の多い感情イベント（年次の敗退）が最も語彙の薄い瞬間になっている。3年連続同回戦敗退は「事件」であり、現状それはリザルト数字でしか起きない。

## 3. 最大の面白さリスク: 停滞レジームの週送りが「義務タップ」化して6〜8時間の中腹で離脱する

- バランスは正しい（初回はほぼ勝てない＝設計どおり）。リスクは数値でなく、**負け続ける5年間を「語られない時間」として送らせる**こと。検証①の不合格基準そのもの（「義務でタップしている感覚」）が、48手×3周でなく本編の4〜7年目で最も出やすい。
- **演出解（1行・規律B-5）**: 支配人／谷口の**年次一言ラダー**を新設する——キーは（年数×最高到達×同一壁連敗数）の3軸だけ、年1回・GP初戦前または年次リザルトに1行差すテキストプール（例: 2連敗「同じ壁は、二度目から名前で呼べ」→4連敗「壁の高さはもう知っとる。今年は測りに行くな、壊しに行け」→8年目裏天井帯「キャリア晩年に一度は、事件が起きるもんや」→10年目「今年が最後や。まだ伸びるぞ」——最終年の行は**両プレイ型で発火**・lastyear_calibration §6の二種の奇跡と接続）。既存の人物・既存の差し込み枠（onboarding §4の支配人年1回登場・ui_design §2-Bの谷口台詞トリガ）を流用でき、新機構ゼロ・在庫は10×5×5でも250行以下で済む。
