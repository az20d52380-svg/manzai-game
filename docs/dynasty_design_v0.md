# 王者編（連覇モード）設計 v0 — 10連覇・飽きられ・返り咲き

- 日付: 2026-07-03
- 位置づけ: 「優勝＝勇退」を拡張し、優勝後に**王者として戦い続ける**エンドコンテンツを追加する。`tools/exp_renpa.py` で全数値を実測済み（各1000キャリア）。固有名は架空、数値は全て【仮】。
- リサーチ根拠: 実在の賞レースでは (a)コア客に愛される実力派が決勝の一般客に届かず準決止まりを繰り返す (b)常連が大量敗退する「波乱の年」がある (c)決勝返り咲きからの優勝は史上2組しかない偉業 (d)連続決勝進出記録やラストイヤー優勝が大会の物語資産になっている——「勝ち続けることの難しさ」自体がジャンルの核。

## 1. モードの流れ

1. グランプリ初優勝時に選択: **勇退**（従来エンド・年表とスタッフロール）／**王者として続ける**（王者編へ）
   - 【v0.3追記】王者編の選択肢は**トロフィー累計18pt以上で解禁**【仮】（`meta_report_v0.md` §1。未解禁のうちは勇退のみ——「王座を守る資格は実績が要る」）
2. 王者編: 王者は予選免除（決勝シード）。ただし**王者ライン**が連覇数で上がる（§2）
3. 防衛失敗 → 王座陥落。翌年は1回戦から出直し（**返り咲き**挑戦。通常ライン95）
4. 勇退条件: 初優勝後に無冠3年【仮】で強制勇退（引き際の物語）。キャリア上限25年【仮】
5. **10連覇達成＝殿堂入り**。専用エンディング（ゲームの最終到達点）

## 2. 王者ライン（飽きられ）【採用: STEP=5・基準94（v0.3改訂・イベント込み正典）】

> 王者ライン = 決勝ライン94 + 5 × 現連覇数（防衛1回目99 → 5回目119 → 9回目139）

物語上の意味: 王者への客と審査の期待は毎年インフレする。「去年と同じ面白さ」は減点である——実在の連覇の壁の機構化。

### 実測（STEP=5・決勝94・イベント層込み正典・上限120固定・25年キャリア・1000試行）※トロフィーはポイント制（trophy_design_v1.md）

| トロフィー累計pt（D） | N相方(5→20) 10連覇率 | SSR相方(20→30) 10連覇率 |
|---|---|---|
| 0pt (D120・初期状態) | 0.0% | **0.0%** |
| 10pt (D130) | 7〜51% | 51〜92% |
| 20pt (D140) | **33〜61%** | 79〜100% |
| 30pt (D150・全解放) | 46〜76% | 91〜100% |

（履歴: v0.2の決勝91基準では 10pt=N10〜63%/SSR59〜95%、20pt=N43〜79%/SSR84〜100%、30pt=N61〜91%/SSR94〜100%。正典が決勝94に上がったぶん全体にやや厳しくなったが設計結論は不変。人間はボットより下手なため実機ではこの表より下がる想定——プレイテストで再計測）

- **10連覇にSSRは必須ではない**（v0からの設計変更）: 全トロフィー＋N相方で約3〜5割、半分のトロフィー＋SSRでも届く。「トロフィーを積む」「相方に恵まれる」「両方そこそこ」の複数ルート。初期状態（0pt）ではどの相方でも不可能
- 当初STEP=6も検証したが、N相方の10連覇が数学的に不可能（スコア上限145 < 要求149）になり「SSR必須」が固定化するため5に緩和。代わりにトロフィー総枠を20→30ptに拡張して段階を細かくした
- プレイの腕は最後まで効く（同条件でボット間に29%⇔50%の差）
- 返り咲き経験は王者編キャリアのほぼ100%で発生——「陥落→出直し→奪還」が王者編の基本リズム。ヘリテージ補正（通算優勝数で返り咲きを難化）も検証したが挙動がほぼ変わらないため**不採用**（複雑さに見合わない）
- 記録画面の主役は**最高連覇数**にする（通算優勝は25年サンドボックスでは16〜18回まで積めるため、あえて誇示しない）

## 3. 挑戦者側の「飽きられ」と返り咲き（王者にならなくても起きるドラマ）

リサーチの「実力はあるのに準決で落ち続け、飽きられ、それでも復活する」を挑戦者側の機構として追加する（ネタ資産システムと連動・周回版）:

1. **客層の二層化【仮】**: 準決勝まで＝コア客（センス・発想の重みを+10%）／決勝＝お茶の間（華・表現の重みを+10%）。「準決の魔物と決勝の魔物は別」——コア向けに尖った育成は準決を抜けやすいが決勝で伸びない
2. **飽きられデバフ【仮】**: 同じ回戦で3年連続敗退すると、その回戦でスコア-3（審査が既視感を持つ）。解除条件は (a)ネタの型を変える（neta_system連動） (b)他大会で優勝する（再評価） (c)2年その回戦に出ない
3. **波乱の年【仮】**: 毎年1/6の確率で「審査トレンド変動年」——全ラインが±3揺れる（年初にNPCが噂として匂わせる）。常連が落ち新星が上がる年を人為的に作る
4. これらの複合で「9年連続準決→飽きられ→型を変えて決勝→返り咲き優勝」という実在型の物語が自然発生する

【設計上の懸念】1〜3は初優勝前の難易度分布も動かすため、実装時は `sim_career.py` に組み込んで初回優勝なし率99.9〜100%が維持されるか必ず再計測（特にデバフ-3はライン-3の感度表がそのまま使える）。

## 4. 王者経済（優勝後に金と力が暴走しない仕組み）

実測: 王者編キャリアの最終所持金は中央値**1.7〜1.9億円**（知名度100→オファー率50%＋防衛賞金）。対策を収入・支出・変換の3方向で:

1. **王者の看板料（支出）【仮】**: 知名度連動の生活費: 4週ごとに 10万 + 知名度×2000円（知名度100で30万）。テキストは「後輩に奢る側になった」「衣装・付き人・移動が全部良い席になっていく」
2. **賞金の使い道イベント（変換）【仮】**: 防衛賞金の半分をその年の内に使い道指定（単独全国ツアー=人脈P化／若手ライブ主催=翌年トレンド予知／貯金）。金→力ではなく**金→物語と人脈P**に変換する出口を用意
3. **アイテム購入制限（遮断）**: 金がいくらあっても強さに変換できない設計（`item_design_v0.md` の使用回数制限・恒久強化非売則）
4. 王者編の収支目標【仮】: 年+800万の黒字を看板料と使い道イベントで年+200〜300万まで圧縮し、25年での余剰を数千万円台に収める（実装後に exp_renpa で再計測）

## 5. 演出との接続

- 防衛戦の決勝は挑戦者側と画面を反転（王者は最後に登場・「王者の孤独」演出）
- 連覇数に応じて審査員コメントに王者フックを追加（例・天堂寺「去年より面白い。それだけが王者の仕事や」／音羽「今年も『今年が一番』って言わせる気？」）
- 殿堂入り（10連覇）: センターマイクを置いて去る後ろ姿→翌年の大会で若手が「憧れの人」として自分たちの名を挙げる→次周へ
