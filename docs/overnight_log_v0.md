6. **衝突回避**: 各commit前に `git pull --rebase`。Macとはファイル分離（Mac=Swift / 私=docs・tools）
7. **法務・市場のリサーチは"弁護士/事実確認に持ち込む一次まとめ"**として扱い、断定しない（出典と確信度を添える）
