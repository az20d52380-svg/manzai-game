## 3. 発見と論点

### 3-1. 大阪戎ライン40は「解釈」で妥当性が変わる
通過率14%（最善戦略）。**「通過＝優勝（賞金獲得）」なら新人コンクール優勝として妥当**な狭き門。**「通過＝入賞・手応え」のつもりなら厳しすぎ**（ライン35で29%が候補）。仕様は通過＝賞金なので前者解釈だが、1年完結のMVPで最初の大会が86%失敗に終わる体験が「面白い悔しさ」か「理不尽」かは実プレイで要確認。

### 3-2. バスvs新幹線のジレンマは強烈に機能している
同条件で通過率7.5%対24.4%、**2万円の差が合格率3倍差**になる。狙った「お金を取るか本番を取るか」は機構として成立。ただしこのままだと「バス＝罠」に近い。救済は既に設計内にある: **大会前週に休養して体力を整えればバスのペナルティは回避可能**（これがプレイスキルになる）。チュートリアル等でこの因果を匂わせるかは要検討。

### 3-3. 【設計上の懸念】GP予選ライン60は1年目ほぼ0%→年末が消化試合化する恐れ
最善でも通過0.2%。**複数年の本編なら「1年目は予選落ちが当然」で正しい**。しかしMVPは1年完結のため、第40週でほぼ確定敗退→第41〜47週が目的を失い、検証①（飽きないか）に直撃する。候補: (a)MVPだけ予選ラインを50前後に下げ「決勝の壁」で跳ね返す (b)予選の段階表示（3回戦まで行けた等）で部分的達成感を出す (c)敗退後の週に来年への意味づけを足す（MVP範囲外）。**実装前にどれかを決めたい**（第6段階・大会の作りに影響）。

### 3-4. 相性成長のTBDは小さな決定ではなく主要バランスレバー
「相性+小【TBD】」の有無で戎通過率が13.8%⇔2.9%と激変（上限20まで育つと実質+15点）。実装時に「なんとなく+1」で入れると難易度が意図せず動く。**成長の有無・上限・1回の量は意識的に決める**こと。

### 3-5. 経済は「破産の恐怖」ではなく「機会費用」で機能している
標準バイト常勤なら年+223万の黒字（生活費は脅威にならない）。一方で稽古全振りは第29週に所持金-40万で**遠征費1万円すら払えず大会に出られない**。つまり緊張の正体は「何週をバイトに捧げるか」: 年間支出約123万に対し初期資金30万→**最低約12週（48週の1/4）はバイトが必要**（有料稽古1回ごとに+1週）。この「1/4税」の感触が面白いかが検証①の核になりそう。

## 4. 仕様の未定義箇所（シミュ実装で判明・Swift実装時に決める）

1. オファーの体力消費（仮: -20で実装）
2. オファー2種（お金/経験）の出現比率（仮: 50/50）
3. 大会週にオファー・突発イベントは来るか（仮: 来ない）
4. エントリー確認（仕様の仮案「2週前」）の週の扱い＝その週の行動は消費するのか
5. 優勝時の知名度上昇量【TBD】（仮: +20）
6. コンビ相性は成長するのか（§5は固定値、§6は相性+小と両論。§3-4参照）
7. 所持金マイナス時に有料行動（稽古・遠征）は選べるか（仮: 選べない＝現金必須）

## 5. 次に回す実験の候補

- 生活費12万/バイト収入-20%など「貧乏圧」を上げた場合の必要バイト週数の変化
- 体力ペナルティ閾値（50/30）とバスの組み合わせ最適化
- メンタル・ブレ幅式の寄与度（メンタトレに8万払う価値が出ているか）
- 突発イベント5種（§14）を組み込んだ場合の揺らぎ
