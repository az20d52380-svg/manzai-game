# 課金価格 提案 v0 — B案（F2P・強さを売らない）の値付け

- 日付: 2026-07-05
- 位置づけ: monetization_decision_v0.md（B案確定）の価格未決部分を埋める提案。全て【仮】・国内モバイル相場観に基づく初期値で、リリース前に最終決定する。原則: **強さは売らない**（売るのは「運の試行回数」「時短・快適」「見た目」「物語」）。

## 1. 通貨と基本レート【仮】

- ゲーム内課金石「**座布団**」【名称仮】: **1個 ≒ 1円** の明朗会計（1:1レートは信頼設計。端数レートで錯覚させない）
- 販売パック: 160円／480円／980円（+5%おまけ）／2,940円（+8%）／4,900円（+10%・最大）
  - **最大パックを4,900円に抑える**（青天井の万札パックを置かない）。「強さを売らない」ゲームで高額パックは不信のシグナルになる

## 2. ガチャ（楽屋名鑑）の値付け【仮】

| 項目 | 価格 | 根拠 |
|---|---|---|
| 単発 | 石150 | 1日1回無料（持ち越し無し）が基本供給。有料単発は「今日もう1回」の衝動枠 |
| 10連 | 石1,400 | 単発比7%引き。SR以上1枠確定【仮】 |
| **天井（30連で最高レア確定）** | **実質4,500円** | 業界標準（数万円天井）より大幅に安いのは意図的——SSRは2体だけ・性能は「相性上限+4」の贅沢品であり、**天井が安いこと自体を良心設計として宣伝に使う** |
| 顔合わせ単発 | 1回無料（確定事項） | |

- 確率表示: SSR 1.5%／SR 8%／R 30%／N 60.5%を**ガチャ画面に常時表示**＋提供割合の詳細ページ（ストア審査・景品表示法対応。コンプガチャ構造は存在しない設計を維持）

## 3. 月額パス「**劇場後援会**」【名称仮・580円/月】

- 内容: 毎日の無料ガチャが**2回に**（持ち越し無しは維持）＋人脈P毎日+5＋限定コスメ月1（後援会員証・楽屋のれん等）
- 狙い: B案の弱点「日次ログインは強いが売上の柱が細い」を、**習慣に寄り添う低額サブスク**で埋める。強さには一切触れない
- 480円でなく580円: 国内の「コーヒー1杯」帯。年6,960円=本作の想定コア層のARPPUの現実的な中心

## 5. 時短・快適（消耗しない買い切りQoL）【仮】

- 新幹線年間パス（ゲーム内金でも買える。石での購入=石600は「ゲーム内で稼ぐ時間の時短」）
- 演出スキップ拡張・オート周回補助（2周目以降のみ有効化）: 無料。**テンポを人質に取らない**（レビュー曲線を守る方が長期の売上に効く）

## 6. 売らないものリスト（明文化・変更禁止）

1. 能力・成長上限・ライン緩和——**いかなる形でも売らない**（体験の根拠が崩れ、実測バランスが全部無効になる）
2. トロフィーpt・人脈P の直接販売
3. スタミナ回復（本作にスタミナ課金の構造自体がない）
4. ガチャ天井の引き上げ・期間限定の排出率操作

## 7. 想定収益構造（粗い見立て・全て【仮】）

- 収益の柱の想定順: ①劇場後援会（サブスク） ②天井までの石 ③DLC事務所 ④コスメ石
- KPI目標【仮】: 課金率3〜5%（強さを売らないゲームの現実ライン）・ARPPU 月800〜1,500円
- 詳細な市場性・リスクはレッドチーム（red_team_v0.md）の「稼げるのか」節に委ねる
