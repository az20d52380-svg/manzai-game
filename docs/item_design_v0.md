# アイテム設計 v0 — ゲーム内通貨・課金・金余り対策

- 日付: 2026-07-03
- 位置づけ: アイテム体系の全体設計。ゲーム内通貨（円）で買う消耗品、メタ通貨（人脈P）、課金の3層。数値は全て【仮】。
- リサーチ根拠: 課金アイテムは実務上 (a)パワー系（勝利に直結・P2W批判の対象） (b)時短系（進行補助・実力差を間接的に広げる） (c)コスメ系（性能無関係・最も受容される）に分類される。買い切り型はこの問題を回避できる。本作は**買い切り＋コスメ/追加シナリオDLC**を基本とし、パワー系課金を絶対に置かない。

## 0. 三大原則（先に結論）

1. **恒久的な強さは金で売らない**（円でも課金でも）。恒久強化の入口はトロフィー（才能解放）と相方だけ
2. **消耗品には年間使用回数制限**を付ける。金がいくらあっても回数は買えない
3. **課金でスコア・D・相性・ラインに触れるものは作らない**（P2W禁止則。売るのは見た目・物語・時間だけ）

### 金余り対策の5措置（まとめ）

1. **年間使用回数制限**（上表）: 所持金1億でも栄養ドリンクは年6本
2. **恒久強化の非売**: 円で買える恒久はICレコーダー1点のみ（教育用の例外）
3. **効果の逓減共有**: 能力系アイテムは稽古と同じ逓減式を通る（高能力ほど効かない）
4. **王者の看板料**: 知名度連動の生活費増（dynasty§4-1）で高知名度期の収入を自動圧縮
5. **無限シンクは名誉のみ**: 青天井に使えるのは寄付（称号・演出）だけ。力は買えない

## 2. メタ通貨「人脈P」（周回間の通貨・金では買えない）

- 獲得: 到達回戦・知名度・完走・全国ツアー等の**行動**からのみ。円からの両替は不可
- 用途: 相方オーディションの候補枠の質（partner_gacha§1）／周回開始時の初期ボーナス選択（初期能力+5〜15＝快適性枠）／封印ネタの引き継ぎ（neta_system連動・周回1本まで）
- 意図: 「金持ちの周回」ではなく「良い芸歴の周回」が次周を強くする。P2Wの内圧（円→力）を構造的に遮断する中間層

## 3. 課金設計（iOS・買い切り基本）

### 3-A. 本体: 買い切り

MVP〜製品版は買い切り1本。ゲーム内通貨の販売はしない（円は稼ぐもの、という世界観自体が商品）。

### 3-B. 追加コンテンツDLC（物語を売る）

| 商品 | 内容 | 価格帯【仮】 |
|---|---|---|
| 相方キャラパック | 新しい相方2〜3人（専用会話3割＋専用アーク。**性能はN〜SR帯**、SSR性能は同梱しない） | 数百円 |
| シナリオパック | 「師匠編」「大阪編」等の連鎖アーク集 | 数百円 |
| 審査員追加パック | 新審査員2人（コメント一式。評価軸は既存7軸の再配合＝バランス非破壊） | 数百円 |

### 3-C. コスメ課金（最受容カテゴリ）

- 衣装スキン（性能なし・§1-Bの衣装とは別枠の見た目のみ）／ネタ帳カバー／出囃子（BGM）／リザルト年表のフレーム
- 命名ルール: 性能ゼロを商品説明に明記（「華は上がりません。あなたの気分が上がります」）

### 3-D. 時短（慎重枠・入れるなら）

- 周回オートチケット（一度クリアした年代の自動進行）——**実力差は広がらない**（本作はPvPなし）ため許容。ただしMVPでは実装せず、要望が出てから
- 【設計上の懸念】スタミナ制・広告・ガチャ課金は本作の「下積みの時間の重み」という主題と正面衝突するため導入しない。相方オーディションを課金ガチャ化する誘惑が最大のリスク（人脈Pの円売りは絶対にしない）

## 3-B. 【2026-07-04追記】課金石（B案確定に伴う第4の通貨）

- 用途は**ガチャ専用**（オーディションを引く）。円（ゲーム内通貨）との交換は双方向とも不可＝ゲーム内経済と課金経済を配管で接続しない
- 三大原則は不変: 課金石で買えるのは「出会いの早さと指名権」のみ。強さ・トロフィー・Dには一切触れない
- 配布: ミッション・実績の初回達成で少量（天井の1/3程度/月【仮】）。「無料でも天井が遠くない」体感を守る

## 4. シミュレーション検証計画

1. アイテム全種を「ボットは使える時に必ず使う」ルールで sim_career に注入し、初回優勝なし率（99.9〜100%）と10連覇率（SSR 24〜52%）の変動を計測。**優勝なし率が98%を切ったら回数制限を絞る**
