# 事務所・冠番組・ファン層設計 v0 — タレント化のジレンマ

- 日付: 2026-07-03
- 位置づけ: 「売れるほどお笑いファンが離れる」現象の機構化＋事務所選択システム（課金オプション候補）＋冠番組。実在事務所の特徴をリサーチし、**全て架空名に翻案**（実名はコード・ドキュメントに書かない）。数値は全て【仮】。
- リサーチ根拠: (a)ネタ番組の減少でテレビ露出とネタ披露が分離し、優勝してもトーク/ロケ主体で「ネタの居場所」がない (b)ファンの細分化（劇場派/動画派/賞レース派/切り抜き派） (c)事務所は「最大手=劇場網・仕事量と引き換えにギャラ激安」「職人系=取り分良し・マナー教育」「自由系=取り分6割・縛りなし」「新興実力主義=業界最高水準のギャラ」等、**取り分と環境のトレードオフ**が明確

## 1. ファン層の2軸【新パラメータ: コア人気 0〜100】

既存の知名度（＝お茶の間人気）に**コア人気（劇場のファン）**を追加する。

| | 上げる行動 | 下げる行動 |
|---|---|---|
| 知名度（お茶の間） | オファー全般・冠番組・大会通過 | （減衰なし・現状どおり） |
| コア人気（劇場） | 営業場数+1・単独ライブ+8・賞レース到達+5・ネタ番組系オファー+3 | **冠番組 -10/年・お金重視オファー -1/回** |

### コア人気が効く場所【仮】

1. **GP準決勝までのライン補正**: −コア人気/20（最大−5）。準決までの客はコアファン——「劇場で愛されている組は予選で強い」（客層二層化 dynasty§3-1 の実装形）
2. **飽きられ耐性**: コア70以上で飽きられデバフ（同回戦3連敗の−3）無効——「劇場が育て続けてくれる」
3. 単独ライブの収益・人脈P獲得の倍率
4. 決勝ラインには**効かない**（決勝はお茶の間の祭典。逆に知名度も決勝ラインに効かない——スコアで勝つしかない場所として聖域化）

### タレント化のジレンマ（狙う体験）

冠番組・バラエティで金と知名度を取るほどコアが痩せ、**予選が弱くなり飽きられやすくなる**。「上手いことやる」の答えは実在の生存戦略と同じ——**テレビで稼いだ金で単独ライブを打ち続け、劇場に帰る**（金→コア変換。item_design§1-Cと接続）。テレビか劇場かの二者択一ではなく、「テレビの金で劇場を買い戻す」循環を作れた者だけが両取りできる。

## 2. 事務所システム（架空6社＋フリー）

結成時（周回開始時）に所属を選択。**性能の縦差ではなく生活様式の横差**にする（EV中立が設計目標）。

| 事務所【架空】 | モデル特徴 | 取り分 | 環境 |
|---|---|---|---|
| 大鳥興業 | 最大手・劇場網 | **3割** | 舞台稽古費半額・営業場数の知名度+1・オファー率+10%・若手ライブイベント豊富 |
| 柊屋芸能社 | 老舗・職人気質 | 6割 | **遠征交通費が全額会社持ち**・ネタ番組系オファー比率高（コアが減らない仕事）・礼儀イベント |
| 気楽舎 | ゆるい自由社風 | 6割 | 行動の縛りなし・オファー率−10%・「相方と過ごす」効果+1（楽屋が和む） |
| 太陽プロダクション | バラエティ帝国 | 5割 | オファー率+15%（ただしタレント系中心=コア減が速い）・知名度成長+ |
| 白虎堂 | ネタ至上主義 | 5割 | 全稽古効果+1・オファー率−15%・大会週メンタル+3（「会長の手紙」演出） |
| 果林カンパニー | 新興・実力主義 | **7割** | 入所条件: 知名度30以上（移籍制）・単独ライブ費用半額 |
| フリー | 無所属 | **10割** | オファー率半減・交通費自腹・舞台稽古+2万（稽古場を借りる）・全て自由 |

- ユーザー原案を反映: 「最大手=ギャラ安いが舞台がある」「職人系=取り分良くて交通費かからない」「フリー=仕事少ないがギャラ10割」
- 取り分はオファー・冠番組・営業収入に適用（大会賞金は対象外【仮】——賞金は二人のもの、という世界観）
- 移籍: 年末のみ可・移籍金なし【仮】。果林のみ条件つき。フリー→事務所はいつでも

### 課金との関係【仮】

- 基本ゲーム: **大鳥興業／フリーの2択**（対極の2つで世界観を教える）
- DLC「事務所パック」: 残り4社＋事務所別イベント各3本＋専用マネージャーNPC。**性能は横差なのでP2Wに当たらない**（item_design三大原則と整合）が、発売前に§4の中立性検証を必須とする

## 3. 冠番組【仮】

- **解禁条件**: 知名度70以上、またはグランプリ優勝（王者編の主要収入源＝dynasty§7-1「王者の中間目標」の回答）
- **年契約制**: 契約中は 年収+500万（事務所取り分適用）／知名度+10/年／**コア人気−10/年**／体力上限−10（多忙）
- 降板はいつでも可（違約金100万・知名度−5）
- 演出: 番組名はプレイヤー命名（デフォルト「（コンビ名）のオールナイト作戦会議」）。年次リザルトに視聴率の推移が載る
- 【設計上の懸念】王者経済（dynasty§4）に+500万/年が乗る。看板料・使い道イベントと合わせて、王者編の年間収支目標（+200〜300万）を再計算すること

## 4. シミュレーション検証計画

1. **事務所のEV中立性**: 各社モディファイアを sim_career に実装し、初優勝中央値が全社±0.5年以内に収まるまで数値調整（「どこを選んでも勝てるが、日々の手触りが違う」が合格ライン）
2. **コア人気の準決補正（−5）**: ライン感度表（exp_endgame）より−5は決勝到達を約2倍にする強さ。コア維持のコスト（単独ライブ年25万〜）との釣り合いを実測
3. **冠番組ルート**: 「6年目から冠+単独維持」ボットを追加し、無冠タレントエンド（金持ち・コア枯れ）と賞レース永住エンドの分布を見る
