# 実績・トロフィー設計 v1 — 難易度別ポイント制（総枠30pt）

- 日付: 2026-07-03（v0を全面改訂: ポイント制・バッドエンド枠・10連覇のSSR必須解除）
- 仕組みの確認: **トロフィーは取るたびに全部恒久加算される**（装備・付け替えの概念はない）。1個ごとに難易度に応じた**才能ポイント（1〜3pt）**を持ち、累計ptがそのまま才能限界Dに加算される（D = 120 + 累計pt、最大150）。能力上限は120で固定——Dが120を超えた分は「上限への到達が速く・確実になる」効果として働く。
- 数値は全て【仮】。固有名は架空。

## 1. バランス根拠（実測・各1000〜2000キャリア）

飽きられSTEP=5（王者ライン=95+5×連覇数）・上限120固定での10連覇率:

| トロフィー | N相方(5→20) | SSR相方(20→30) |
|---|---|---|
| 0pt (D120) | 0.0% | **0.0%**（初期状態では誰でも不可能） |
| 10pt (D130) | 1〜23% | 34〜82% |
| 20pt (D140) | 20〜41% | 65〜98% |
| 30pt (D150) | **29〜50%** | 81〜100% |

- **SSR必須ではない**: 全トロフィー＋N相方でも約3〜5割で10連覇に届く。逆に、トロフィー半分＋SSRでも届く。「トロフィーを積むか、相方に恵まれるか、両方そこそこか」の複数ルート
- 初回体験は不変（優勝なし99.9〜100%・決勝到達17〜28%を再確認済み）
- ボット間の差（例: 30pt N相方で29%⇔50%）＝プレイの腕は最後まで効く
- 130→140→150の伸びが逓減する曲線＝「後半のトロフィーほど貴重だが、取り逃しても詰まない」

## 3. 取得カーブ（周回ペーシングの検算）

- **初回**: #1,2,7,9 ほぼ確実＋#3,5,10,13 高確率＋#12,15,16,18 のどれか → **8〜11pt**（D128〜131）
- **2周目**: 王者編解禁帯（D130前後）。SSRを引ければ10連覇チャレンジ権、Nなら「連覇7〜9の壁」を体験
- **3〜4周目**: 大会制覇系と2pt帯を回収 → 20pt台（D140台）でN相方でも10連覇が現実に
- **最後まで残る**: #21（奇跡）と#22（六冠）——ここだけは運と完璧なキャリアの両方が要る

## 4. 設計ルール

1. **総枠30pt・D上限150は絶対固定**（インフレ運用禁止。イベント報酬でptを配るなら本表から置き換える）
2. バッドエンド枠（#18,19）は「負けても前に進む」の機構化。周回ゲームでは**失敗も収集対象**にする
3. 取得済みトロフィーの再取得は無効（周回で同じ実績を再達成してもptは増えない）
4. D加算の適用は周回開始時（キャリア途中の取得は次周から）——シード再現性の維持
5. 称号（0pt・15種）は v0 §2 のまま別枠で存続
