# 会話量産バッチ2 v0 — 日常・バイト・効果つきミニイベント

- 日付: 2026-07-04
- 位置づけ: `dialogue_design_v0.md`（基本トーン・相性帯差分）の増産巻。バイト現場の会話イベント、日常雑談ローテ、**数値が動く選択会話**、予約仕事（booking_design_v0.md）の会話全量。固有名は架空、数値は全て【仮】。
- 執筆原則（会話劇の技法リサーチより一般化）:
  1. **全ての会話に「なぜ今この話をするか」の目的を置く**（雑談に見えて、どちらかが何かを欲しがっている）
  2. **オチは基本、谷口（相方）が持つ**。主人公は受け手＝プレイヤーの視点
  3. **沈黙・目線・行動で語る行を1本に1つ**（喋らない演出に感情が乗る）
  4. 関係・感情・伏線に奉仕しない台詞は削る。1イベント最大6行
  5. 日常の手触りを丁寧に積む——だから異変（解散危機・敗北・優勝）が効く

## 2. 日常雑談ローテ（自由週の結果画面・低頻度差し込み・効果なし〜微小）

関係を積むだけの「無駄に見えて無駄じゃない」枠。1本2〜3行厳守。

1. 谷口「コンビニの新作、ネタになるかと思って買ったら普通にうまかった。以上です」「報告の意味よ」
2. 銭湯の帰り道。どちらも喋らなかった。それが嫌じゃなくなっていることに、ふと気づいた。（相性10以上で表示）
3. 谷口「なあ、優勝したら何買う？」「先に決勝行け」「夢の順番の話してんのに、現実の順番で返すな」
4. 家賃の振込を忘れかけた。谷口から先にメールが来た。『家賃』の二文字だけ。保護者か。
5. 谷口「昨日のテレビ見た？」「見てない、ネタ書いてた」「……正解。俺は見た。面白かった。**悔しかった**」（メンタル+1）
6. 傘が一本しかなかった。「入れよ」「濡れるやろ」「半分こだよ」——世界で一番安い漫才の稽古。（相性+1・年1回）
7. 谷口が財布を拾って交番に届けた話を、3日連続で聞かされている。「いい話は一回でいい話なんだよ」
8. 深夜のファミレス。隣の席の大学生が俺たちのコンビ名を言った。サインは求められなかった。**まだ、その距離**。（知名度15以上で表示）
9. 谷口「お前、寝ろよ。目の下すごいで」「ネタ直したら寝る」「……ほな俺も起きとくわ。監視な」（大会前週のみ）
10. 母から荷物。谷口の分の米も入っていた。あいつ、いつの間にうちの実家と繋がったんだ。

## 4. 予約仕事の会話全量（booking_design_v0.md 連動）

- **予約受諾**: マネージャー「3週後の第◯週、収録です。**入れますね？**」／谷口がカレンダーアプリを開く。俺は壁のカレンダーに書く。アナログとデジタルの二重チェック体制、結成以来無遅刻。……今のところ。
- **前週の伏線**（さりげなく・モーダル禁止）: 谷口「来週やで、例の仕事。……俺が覚えてるの、逆に不安なるやろ」
- **当日・行く**: 現場のADさんが台本をくれた。俺たちの名前が印刷されている。**印刷された名前は、手書きの100倍プロっぽい**。
- **バックれ翌週**: マネージャーの電話は静かだった。「先方には私が謝りました。……次は、電話が鳴らないだけです」／切れたあと、谷口は何も言わなかった。何も言われないのが、一番堪えた。（知名度-3・オファー率半減8週の明細表示はこの直後）
- **遅刻強行（体力20未満で出勤）**: 楽屋の鏡に映った顔がひどい。ディレクター「……来ただけ良しとします。**次はちゃんとした顔で来てください**」／ギャラ明細は半分。谷口「顔で半額て。俺らの顔、定価いくらやねん」——落ち込む間も、こいつはボケる。
- **大会と重複（自動キャンセル）**: マネージャー「大会ですから、先方も『そっち勝ってこい』と。**勝ってきてくださいね、貸しですよ**」

## 5. オファー現場スケッチ（既存2種の「行った先」を初めて描く）

- **お金重視（営業）**: ショッピングモールの特設ステージ。客は買い物のついで、前列は子供。ウケるかは運。終演後、子供がひとり寄ってきて「もういっかいやって」と言った。**アンコールの最小単位**。（華+1の隠し効果はこの演出時のみ【仮】）
- **経験重視（ネタ番組の前説・エキストラ枠）**: 本番前のスタジオは寒い。プロの現場の「段取りの速さ」を体で覚える。放送に俺たちは映らない。でも、**次に呼ばれる人間の顔**は覚えてもらえた。
