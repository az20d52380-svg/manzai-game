## 5. 実装メモ

- 変な仕事はオファーテーブルの差し替えで実装（GameEngine.rollOffer の手前で知名度分岐）。乱数消費順序が変わるため、**実装時は gen_golden.py と CareerGoldenTests の再生成が必須**（CLAUDE.mdルール5）
- R2再エントリーは1回戦の週構造に手が入る（第32週に予備週）。sim_career に入れて初回優勝率1.5〜2%が動かないか要再計測（+0.3%程度の見込み）
- 効果つきイベント追加後は `sim_career.py` のイベント層再検証（event_design_v0 §5）とまとめて回す
