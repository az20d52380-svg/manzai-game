## 2. v2で成立する体験（実測サマリ・準決74/決勝80・n=1000）

| プレイスタイル | 初回優勝（10年内） | 決勝到達 |
|---|---|---|
| やり込み（分散稽古＋金・体力管理） | 2.1%（中央値10年） | 43.5% |
| のんびり改（初心者＋大会前に休むだけ） | 0.3% | 21.7% |
| 金策放棄・不休の縮退戦略 | 0%（破産・ダウンで退場） | - |

- 「初回はほぼ勝てない・決勝の悔しさは2〜4割が味わう・準備の腕が効く」が**全プレイスタイルに対して**成立（旧: 特定スタイルで優勝100%）
- ここにトロフィー（上限カーブ持ち上げ）と相方ティアが乗って周回で優勝が現実化する（§4の再スキャン参照）

## 2-B. 【2026-07-05・重要】メタ報酬の再価格が必要（exp_v2_meta の発見）

旧正典の周回報酬をそのままv2に載せると壊れることが実測で判明:

| 旧報酬 | v2での実測 | 対策と再実測結果 |
|---|---|---|
| 初期能力+20（周回ボーナス） | 分散型の初回優勝が2.1%→**63%**に暴騰 | **✅解決: キャリア累計予算化**（`sim_career.CAREER_BUDGET`・初期ボーナスは予算の前借り＝テンポ報酬）→ 2.8%に復帰 |
| SSR相方（相性20→30） | のんびり改ですら0pt優勝**78%** | **✅解決: ティア再価格（SSR=相性上限+4）**→ 分散13.3%・のんびり改4.6%。「SSRを引けば初回でも夢がある」の贅沢品として機能（微調整はcap22〜24で可能） |
| トロフィー効果 | （旧D加算は廃止） | **✅成立: 上限カーブ+0.02×pt/年**で 0pt=2.8% → 11pt=6.2% → 20pt=13.2% → 30pt=**21.6%**（分散型・N相方）——周回で優勝が現実化する本命カーブ |
| 王者編STEP | 成長15年終了（`GROWTH_END_YEAR`）で怪物化は解決。ただし**STEP2でも10連覇0%**——連覇10回はハマった夜の連鎖を要求してしまう | **🔄要再設計（v2ネイティブ）**: 単純移植は不可能領域。候補: (a)王者は成長制限を超えて伸び続ける（王者の特権） (b)防衛成功ごとにハマ率+α（王者の風格） (c)防衛戦を別形式（相対評価）に。次バッチで設計＋スキャン |
