**【2026-07-05補記・オーナー確認】「決勝進出は運か？」への答え: 内部スコアそのものが「ウケ量」である。** 実力（積み上げ）＋相性＋出来（恒常ブレ）＋その夜の客席との化学反応（ハマった夜）＝その日のウケ量、がラインと比べられる——実在の予選審査（ウケ量メイン＋構成＋人気）と同型。運だけでは絶対に上がれない（実力が足りない組はハマっても届かない）し、実力だけでも上がれない（準決ラインは恒常ブレの外）。**実力上位はハマらなくても通り、ボーダー勢はハマった夜に通る**——実在の準決レポの景色と一致する。ハマは「見える運」としてファンレポ（fan_report_design §1）と笑い波形で必ず可視化する。未決: 何年もハマらない人の裏天井（例: 3年連続ハマなしで翌年率2倍【仮】）。将来はネタ補正（±5・完成度=プレイヤーが管理するウケ量）が乗り、運の比重はさらに下がる
