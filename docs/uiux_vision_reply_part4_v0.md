## 依頼2：画面ごとの詳細UI/UX仕様

### S11 アイテムショップ

**レイアウト**
- S2コマンドゾーンの場所がそのまま**商店**になる（別画面に飛ばない=浮かない）。カテゴリ「アイテム」タップ→カード列の位置に商品カード横スクロール＋上に**棚タブ3つ**: 「コンビニ・商店街」（序盤帯）／「投資」（中盤帯）／「王者の出口」（王者帯・王者編以外は切り絵シルエットの棚=まだ出会っていない棚）。
- 商品カード=変種カード構造の流用（帯色・絵柄・コスト行=¥等幅）。ただし実行でなく購入なので、カードに**回数券の綴り**を常設——年間制限ぶんのミシン目付き小片（栄養ドリンク=6片・カツカレー=大会ごと1片…）＋等幅「あと◯回」。**購入ボタンの隣に残数**（確定要件）。寄付（無限シンク）だけ綴りが無い（青天井=見栄、を綴りの不在で示す）。
- 棚の上に貼り紙1枚: 「強さは、置いていません。」（三大原則の店内掲示。恒久はICレコーダーのみ=購入すると棚から**自分の机**へ移り、以後は商品でなく持ち物として描く）。

**トランジション**: カテゴリ切替=S2既定の同位置切替0.25s（matchedGeometry連結）。棚タブ=クロスフェード0.2s。購入確定=カードが**紙に包まれ紐が掛かる0.3s**（キツ入り緩抜け）→所持金`numericText`0.3s＋「−◯円↘」差分ポップ。週は進まない（裁定1）。
**マイクロインタラクション**: 購入で綴りが1片**切り取られる0.2s**（ミシン目の走り）——「使い切る文化」の物性の本体。即時効果系は該当ピルに+N（既存規格）／「その週」系はカード上に当週バッジ。残高不足・回数切れ=沈まず横ブレ0.15s＋トースト（不足=既存文言／切れ=「今年の分は、使い切った。」）。効果量は事前表示しない既定を維持（逓減も含め、買って使って+Nで分かる）。
**音・ハプティクスのキュー**: 購入=紙包みSE小のみ・**無振動**（頻回操作に振動を付けない規約。例外はICレコーダー購入=1回限りの節目のみ`hConfirm`）。棚タブ・スクロール無音。
**実装ブリッジ**: 流用=変種カード／トースト／差分ポップ／numericText／シルエット様式（王者棚）。新規=棚タブ・綴りUI・包み0.3s（**購入の適用タイミング・回数カウンタはGameCore側と要すり合わせ**）。難度: **中**。

### S12 図鑑・記録

**レイアウト**（縦1カラム・上から）
1. **記録の帯**（第1便§4-4確定）: **最高連覇数を主役に等幅大**・最短優勝・総獲得賞金。通算優勝は小さく添える（誇示しない=`dynasty` §2確定）。
2. **トロフィー棚**（22個/30pt・実物棚メタファ——S6の資格演出で見せた棚の本体）: 取得済=実物イラスト小／未取得=切り絵シルエット＋条件1行。**バッドエンド枠（#18十年選手・#19雨天決行）も同格に並ぶ**（失敗も収集対象=`trophy` §4-2。棚の中で区別しない）。ptは各札の隅に等幅小字。
3. **称号の列**（0pt・15種・木札様式=S8天井の木札と同部品）: 「白票の夜」等。
4. **通算年表**: 全周回1周=1行（紙芝居プレイヤー流用・行タップで年表詳細へ）。各行末に**その周の最終レーダーのサムネイル**（レーダーCanvas流用）——長押しで現在周と重ね比較0.6sモーフ（比較対象は常に過去の自分=§4-3原則の周回版）。
5. 歴代コンビ写真の小列（壁写真層のデータ流用・タップでS8名鑑の該当カードへ）。

**トランジション**: S2「データ」からは既定の同位置切替0.25s／S7からはクロスフェード0.18s。棚へのスクロールは慣性のみ（記録の画面は静かに・演出を足さない）。
**マイクロインタラクション**: トロフィー取得の瞬間は取得場所（大会中等）で判+`hConfirm`の再利用（第1便§4-4・新文法を増やさない）。S12側は**新規取得札に朱点**が付き、開けば消える（未読管理・パルスさせない）。未取得シルエットのタップ=条件1行のトースト（取得率%は出さない——他人の達成度で煽らない）。
**音・ハプティクスのキュー**: 開閉の紙めくりSEのみ・全閲覧無振動。
**実装ブリッジ**: 流用=**共用部品3点すべて**（レーダーCanvas・紙芝居プレイヤー・壁写真層）＋判＋木札。新規=棚グリッド＋未読朱点の永続化（**周回間永続は`dynasty` §6のリスト——最高連覇・通算優勝・返り咲き回数・殿堂入りフラグ——と要すり合わせ**）。難度: **中**。

## 依頼7：S13 将来ビジョン（自由記述・投機枠——採否はオーナー判断）

既出チェック済み: 通知=S1c／デイリーミッション=S7／ピックアップ告知=S7ポスター／データ管理・クラウドセーブ入口=S1b。以下は未出のみ。各項に主題（下積み・実力主義・無料で全部遊べる誠実さ）との整合を1行で自己チェックする。

1. **業界紙（統計の読み物・ランキングの翻訳）**: 週1回、「全国の楽屋」の匿名集計を紙面様式で——今週の決勝到達率の分布・稽古傾向の円グラフ・自分は分布の中の点1つ。順位表と他人の名前は出さない。〔自己チェック: 「比較対象は常に前の自分」原則と両立——分布は風景であり順位ではない〕
2. **拍手だけのフレンド（合いカギ）**: コード交換で相手の楽屋の壁（完走写真）を見学し、拍手1種を1日1回置けるだけ。数値報酬なし。〔自己チェック: 報酬ゼロなので義務化せず、対戦なし原則も維持〕
3. **回覧板（お知らせ）**: S7の壁に紙1枚が留まる（赤丸1つ・カルーセルバナー禁止・タイマー禁止）。月1体のキャラ供給ロードマップ（`monetization` §7懸念1）の告知が主用途。〔自己チェック: 運営告知を煽り様式に翻訳しない〕
4. **番組スチル共有**: リザルト・決勝確定の瞬間を1枚画像（コンビ名+判+講評1行+年表断片）に書き出してSNS共有。`monetization` §7懸念5「切り抜き適性が露出戦略の軸」の実装先。〔自己チェック: 送信はユーザー操作のみ・誠実な露出〕
5. **番組表（シーズンパスの翻訳・慎重枠）**: 四半期の無料ミッション束+称号。有料トラックを置くなら**コスメのみ**とし、シーズン終了後は定番として人脈P交換所に落とす（FOMOを構造的に外す）。〔自己チェック: 三大原則とは両立するが「期間限定」自体が煽り圧を持つため、導入判断そのものを慎重枠とする〕
6. **Game Center連携**: トロフィー22個のOS実績ミラー＋クラウドセーブの実装先（S1b入口の裏側）。〔自己チェック: 既存機構の露出のみ・新規報酬なし〕

**提案しないもの**（理由明記）: 対人ランキング/PvP=「相手は自分の過去記録のみ」と正面衝突／期間限定ガチャ=天井+ピックアップ+指名で足りておりFOMOを増やすだけ／スタミナ制=週送りテンポが商品そのもの。
