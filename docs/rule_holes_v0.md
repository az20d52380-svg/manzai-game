# ルールの穴 総点検 v0 — 借金・体力・その他の悪用可能点と対策設計

- 日付: 2026-07-04
- 位置づけ: human_calibration_v0.md の発見を受けた、行動ルールの穴の全数点検。他ゲームの前例をリサーチし、対策を設計してsimで効き目を実測した（検証フラグは全て既定OFF・goldenに影響なし）。数値は全て【仮】。

## 0. リサーチ — 他ゲームはどうしているか

- **低体力練習のケガ制（野球育成SLGの標準）**: 練習ごとにケガ率が表示され、0%以外は常にリスク。軽傷=経験値低下・重傷=能力低下。「ケガ率は表記通り」であることが検証されコミュニティの信頼になっている → **確率は正直に表示する**のが定石
- **借金の扱い**: すごろく系=資産マイナスで物件強制売却（持ち物で払わせる）／商店経営ADV=借金返済そのものをゲーム目標化／RPG=敗北時に所持金半減。共通するのは「**マイナスは許すが、放置すると持ち物・行動・物語で払わされる**」——即ゲームオーバーにはしない段階設計
- 実績デザイン論より: 崖（即死ルール）は事前警告とセットでないと理不尽になる → 段階制＋予告演出

## 2. 対策① 借金3段制【H1・実装推奨】

「借金即死」は下積み芸人の世界観に合わない（借金エピソードは芸人文化の定番）。**許すが、段階的に生活・持ち物・物語で払わせる**:

| 段階 | 条件 | 効果【仮】 | 演出 |
|---|---|---|---|
| 1 生活苦 | 所持金 < 0 | 稽古の能力上昇 ×0.5〜0.75（空腹は芸の敵）＋生活費徴収時に体力-10・メンタル-3 | 会話が貧乏帯（dialogue_batch3 §1）に切替・ヘッダの所持金が赤 |
| 2 督促 | 所持金 < -50万 | 4週ごとに督促イベント: 「強制バイト週（行動固定）」または「質入れ（コスメ・アイテムを売却して+n万）」の二択 | 大家の春さん・谷口の督促会話。**破産まであと◯万の予告**（崖の事前警告） |
| 3 夜逃げ | 所持金 < -100万 | **キャリア強制終了（バッドエンド）**。周回へ。称号「夜逃げ同然」(0pt)＋専用リザルト | roadmapの「借金破綻エンド」の正式化。翌周は谷口の「もうあんな夜はごめんやで」会話で開始 |

### 実測（1000キャリア・イベント込み正典・`tools/sim_career.py` の DEBT/BANKRUPT フラグ）

| プレイスタイル | 対策なし | 借金3段適用後 |
|---|---|---|
| 鉄人型（不休・金策なし） | 優勝100%（4年） | **破産100%・優勝0%** |
| 稽古全振り（金策なし） | 優勝100%（5年） | **破産100%・優勝0%** |
| バランス型 | 優勝1.3% | 1.3%（無影響） |
| のんびり型 | 優勝1.9% | 2.5%（誤差内・破産0%） |

**金策を放棄する縮退戦略は完全に閉じ、普通のプレイは最低所持金-3万程度で段階1にすら滅多に触れない**——狙い通りの非対称な効き方。

## 3. 対策② 低体力の体調システム【H2・v0.1改訂: 方式比較の上ハイブリッド採用】

**用語決定【2026-07-04オーナー指示】: 「ケガ」という語は漫才師に合わないため使わない。** ゲーム内表記は「**喉をやられた**」「**腰にきた**」「**熱でダウン**」等の体調語で統一（総称: 体調ダウン）。

### 方式比較（リサーチ＋実測 各1000キャリア）

他ゲームの前例: ウマ娘=失敗率表示のリスク制（失敗で能力上がらず・稀に能力低下。ただし失敗率への不満記事も多い）／野球SLG=ケガ率表示のリスク制／恋愛SLG古典=無理を続けると病気で数週離脱。**ハードゲート（押せない）は金銭系では標準だが体力系では少数派**——ただし本作には「相方がいる」という固有の正当化がある。

| 方式 | 鉄人型(不休)への効き | 通常プレイ | 所感 |
|---|---|---|---|
| A 体調ダウン制（療養3週+メンタル-5） | ダウン22.7回/人・初優勝5年に後退 | 無影響（ダウン0回） | 緊張感はあるが乱数の理不尽感（実在ゲームでも不満の火種） |
| B A+能力-2（重症変種） | ダウン26.4回・初優勝6年 | 無影響 | **効果がほぼ増えない**のに理不尽感だけ増える → 不採用 |
| C ハードゲート（体力20未満は稽古不可） | 実力への影響ほぼなし | 無影響 | 最もシンプル・3秒ルールと好相性・乱数ゼロ |
| D **ハイブリッド【採用】** 稽古=ゲート・キツいバイト=ダウン制 | Cと同等 | 無影響 | 下記 |

### 採用: D ハイブリッド

- **稽古はゲート**: 体力20未満で稽古ボタンがグレーアウト。理由づけは世界観そのもの——**「谷口が止める」**（「今日はやめとこ。相方倒れたら、俺明日から一人で何すんねん」）。コンビものだから成立する制止であり、モーダル説明が不要
- **キツいバイトはダウン制**: 生活のために無理する自由は残す（onboarding §5-3「倒れる自由」）。選択時にダウン率を正直に表示（⚠️12%——「表記通り」が信頼を生む）。ダウン=療養3週＋メンタル-5、演出は「腰にきた」等でバイト会話（K5系）と接続
- 能力低下（B案）は不採用: バランス実測でほぼ無意味＋演技能力がバイトで下がる理不尽さ。**療養3週の機会損失が実質の能力コスト**として既に機能している
- どの方式もバランス単独では縮退を止めない（それは借金3段と成長上限の仕事）→ 体力システムは**体験・世界観の選択**であると実測で確認した上でDを採用

### 実測（従来のまとめ）

- 体調システム＋借金3段のセットで、金策放棄系の縮退戦略（鉄人型・稽古全振り）は破産100%で完全閉鎖。通常プレイは全方式で無影響

## 4. 残る本丸と、対策の全体像

**この2つは「生活ルール」であり、金管理する分散稽古型（優勝100%・human_calibration §2）は止められない。** 全体像は二層:

1. **生活ルール（本書・即実装推奨）**: 借金3段＋ケガ制。露骨な悪用と非現実的な挙動を塞ぎ、物語（督促・夜逃げ・故障）も増える。normalプレイへの副作用は実測ゼロ
2. **成長経済（human_calibration §3・オーナー判断待ち=master_spec未決8）**: 年間成長上限への移行。分散稽古の構造優位そのものを消す本丸

## 5. 実装メモ

- **【重要】GameCore反映時は乱数消費が変わる**（ケガ抽選が正典順序に入る）→ `gen_golden.py` の正典順序更新と golden 全再生成・Swift同期が必須（CLAUDE.mdルール5）。**MVP実装のタイミングで2対策を一緒に入れて、golden再生成を1回で済ませるのを推奨**（1年版でも軽い赤字と低体力は普通に起きるため、MVPでの体験価値もある）
- H3（本番ペナルティ3段目: 体力10未満で-15【仮】）はライン感度に触るため、生活ルール導入後に再計測してから判断
- H7（東京大会のエントリー費1〜3万【仮】）は小穴。未決リストへ
- 検証フラグ: `sim_career.py` の DEBT_LIFE_PEN / BANKRUPT_LINE / INJURY_* と `balance_sim.py` の DEBT_TRAIN_FACTOR（全て既定OFF・goldenバイト一致確認済み）
