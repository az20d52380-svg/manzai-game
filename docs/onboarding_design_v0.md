# オープニング〜1年目 オンボーディング設計 v0

- 日付: 2026-07-03
- 位置づけ: 初回プレイの導入台本。目標は「最初の8週で全行動を1回ずつ自然に踏ませ、第12週の初大会で『大会の型』を学ばせる」。説明文でなく状況で教える。固有名は架空、数値は全て【仮】。

## 1. オープニング（3分・スキップ可）

1. **回想**: 高校の文化祭。ウケた記憶のフラッシュ（静止画3枚+歓声SE）。「あの4分のせいで、人生を間違えた」
2. **上京**: 家賃3万のアパート。隣室から壁越しに聞こえるネタの独り言——谷口も同じ間違いをした側だと分かる導入
3. **結成**: ファミレス6時間（event#17の原点）。コンビ名入力→「よし。……で、どうやって売れるんだっけ」
4. タイトルロゴ→第1週へ

## 3. 第12週・初大会（春新人賞A）＝「負け方」のチュートリアル

1年目の実力値は20前後、ライン55——**確実に負ける**。ここで教えるのは大会のUIと、負けても物語が続くこと。

1. 遠征の選択（バス/新幹線）を初体験。所持金的にバス一択に見える状況を意図的に作る（夜行バスの車内会話つき）
2. 本番→紙講評（必ず「直せる一点」入り。judge_design §10-A）
3. 帰り道の会話: 谷口「初めて『出場者』って呼ばれたな」——敗北を出発点として記録
4. 直後に実績システム初解放の通知（「初舞台の足音」はGP1回戦なので、ここでは称号側の演出のみ）

## 4. 1年目後半の里程標

- 第29週・大阪戎（ライン40）: **初めて勝てるかもしれない大会**。通過すれば初賞金演出（手元10万の使い道を谷口と話す会話）
- 第30週〜・グランプリ1回戦: トロフィー#1「初舞台の足音」がほぼ確実に取れる→**才能解放の演出を1年目のうちに1回見せる**（周回システムの存在を予告）
- 第47週: 決勝をテレビで見る側の演出（finals_direction §0）「来年は、あっち側だ」
- 大晦日(#季節48)→年次リザルト→2年目へ

## 5. 禁止事項（チュートリアルの毒）

1. モーダルの連続説明ダイアログ禁止。教えることは1週に1つ
2. 「おすすめ行動」表示禁止（improvement §4）。導線はNPCの台詞とカーソル初期位置だけ
3. 失敗の先回り禁止。体力0でバイトして倒れる自由は残す（倒れたら event#16 系の救済テキスト）
