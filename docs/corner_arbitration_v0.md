## 0. 一行結論（【仮】）

> **角は必勝化しない（規範ii ✓・分散角60.75%<90%・多シード60.2〜61.4%で頑健）。火力はほぼ全て30pt×SSRの対に集中し、後段3レバー（裏天井/得意A/絆）は角の中で寄与≒0＝約61%でプラトー（規範iii ✓）。0pt初回帯は不変（規範i ✓）。角スコープ補正は現時点で不要——ただしMac本実装後に4.8が正典ハーネスで角セルを再測することを条件とする。**

## 1. 裁定（規範判定）

### 裁定A: 必勝化せず（規範ii ✓）

- 角セル（全部ON・やり込み）: **優勝60.75% / 到達99.0%**（n=800）。多シード n=500×5: **60.2〜61.4%に密集・全シード<90%**。
- のんびり角: 優勝7.00% / 到達76.5%（多シード6.0〜10.2%）。SSRとトロフィーを全部積んでも、稽古を割かなければ勝ち切れない（sideways維持・「ガチャで勝てる」にならない）。
- 到達99%/優勝61%のギャップ自体がテクスチャ: 最強状態でも10年で4割は無冠＝決勝の夜の特別さは角でも残る。

### 裁定B: 加算的（弱い超加算）でプラトー・乗算暴走なし（規範iii ✓）——ただし安全機構は「二重の壁」

積み上げ（分散・n=800）: 41.4%/2.62% →(+30pt) 89.0%/27.25% →(+SSR) 97.9%/**60.88%** →(+裏天井) 99.1%/61.25% →(+得意A) 96.9%/58.88% →(+絆) 99.0%/**60.75%**。

- marginalΔ優勝: 30pt **+24.62** / SSR **+33.62** / 裏天井 +0.38 / 得意A −2.38 / 絆 +1.88。**後段追加（角−30pt+SSR）= −0.12pt ＝完全飽和**。
- 合成Δ/Σ単独 = 1.56（弱い超加算・30pt×SSRの対で強化）。乗算的暴走なし。
- **言語化（本裁定の追加）——飽和を支える壁は1枚ではなく2枚**:
  1. **予算クランプの壁（分散＝稽古する型を守る）**: 得意Aの+20%は`add()`の年間成長予算クランプに丸ごと吸われ（単独Δ+0.50・角内マージン−2.38＝ノイズ）、絆の+8相性はcompat稽古で到達済みの上限24に飲まれる（角内マージン+1.88）。稽古量が多いほど後段レバーは冗長になる。
  2. **ライン距離の壁（のんびり＝稽古しない型を守る）**: のんびりの角は超加算（Δ/Σ=5.89・単独はほぼ全て0）——各レバーが別々のボトルネックを外すため合成で初めて効く——**が、絶対値は+6.62ptで優勝7%止まり**。守っているのはプラトーではなく「実力ラインまでの距離」。
- 含意: 壁1は**本実装の配線に依存**（得意A/絆が予算・上限の内側に実装されることが前提＝§4）。壁2は**フラットな実力直付与レバーの追加に弱い**——のんびり角で最大の単一寄与は絆の+4.62pt（積み上げ最終段）であり、**絆は角の複利の最有力侵食ベクトル**（機構が最も薄い＝最も仮、という叩き台の評価と整合）。将来レバーを足す際は「稽古量を要求しない直付与か？」を角チェックの第一問にすること。

### 裁定C: 0pt初回帯は不変（規範i ✓）

全レバーOFF: のんびり22.9%/0.38%・分散41.4%/2.62%（canonical §2: 23.1%/0.3%・41.5%/2.7%とノイズ内一致）。

### 裁定D: 角スコープ補正は現時点で不要（条件付き・4.8裁定を維持）

- 60.75%<90%のため `fable_findings` Q5 の (a)能力別合計効率キャップ／(b)SSR角犠牲 は**いずれも今は機構化しない**（config を先回りで増やさない）。
- **もし本実装後の再測で必勝化の兆し（角>80%台後半 or 後段追加が+3pt超）が出たら、第一推奨は(a)合計効率キャップ**（得意+絆+裏の恒久/一時をすべて加算合成・乗算禁止でキャップ）。理由: 現在の無害性が壁1（予算クランプ）の配線依存であるため、壁が壊れた時に壁ごと直すのが(a)であり、(b)は症状（SSR角）しか叩けない。
- のんびり角の監視規範を追加提案【仮】: 正典再測時に「のんびり角優勝 ≤15%」を追加の見張り線とする（現状7%・多シード最大10.2%。壁2の侵食＝直付与レバー混入の早期検出器として機能する）。

## 4. 「仮の仮」の明示と再測条件（裁定に刻む・キックオフ禁止事項5）

本裁定は以下の**実験的近似の上での方向確定**であり、Mac本実装後に4.8が正典ハーネスで角セルを再測するまで確定しない:

1. **得意A**: `do_training`ラッパで演技系4能力×1.20のgenerous版（上バウンド測定・mental/compat/stamina/fame漏れゼロは実測保証済み）。**run_year未配線**。再測の焦点: 本実装が効率+%を`add()`の予算クランプの**内側**に置くか（壁1の存続条件）。クランプを迂回する配線（例: 予算消費後の直加算）になった場合、飽和の根拠が崩れるため角セル再測を実装コミットと同時に行うこと。
2. **裏天井**: BURST_P一時上書き＋`_gp_perform`ラップのMODE C近似。**run_year未配線**（pity_calibration §5-1の仕様で実装）。
3. **絆**: compat_start底上げ（+8【仮】）近似＝**角の中で最も機構が薄い**。値がプレイ型と上限力学に完全依存（分散はcap24に飲まれ≒0・のんびりは決勝まで生き+4.62）。機構の採否そのものを含めオーナー/設計判断が残る。採用時は壁2の侵食ベクトルとして§1-Dの見張り線とセットで。
