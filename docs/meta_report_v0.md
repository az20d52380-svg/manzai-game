## 3-B. 【v0.1追記 2026-07-04】ティア到達を名鑑所有モデルに差し替え（B案対応）

旧「人脈P閾値でティア解放」を廃止し、実際のガチャ（日次無料7回/周【仮・ミドル層】＋人脈Pチケット・気になるリスト・ダブり還流）で相方を所有するモデルで再計測（sim_meta GACHA_MODE）:

- SSR所有率が段階的に広がる（2周目20%→5周目50%→10周目80%）——閾値モデルの「全員一斉SSR」より自然な分布
- 殿堂カーブは従来帯を維持（バランス型: 4周目41%→10周目92%／調整型: 12周で60%）
- **設計注記**: モデル上の「1周目優勝3〜6%」は周回前にSSRを引いた層を含む計算上の産物。実装では**初回キャリアは谷口固定（相方オーディションは2周目から解禁）**とし、初回優勝1%強の設計を守る。これはオンボーディング（谷口が初回の物語の核）とも一致する

## 4. 残課題

1. 事務所2社（白虎堂・柊屋）の微調整（§3）
2. 「雨天決行」（解散危機アーク・1pt）はイベント未実装のためシミュ対象外——実装後にptカーブへ+1される影響は誤差の見込み
