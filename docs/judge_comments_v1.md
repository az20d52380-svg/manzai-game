# 審査員コメント量産 v1 — 実在講評の型分析と予選紙講評・決勝コメントの増量

- 日付: 2026-07-04
- 位置づけ: `judge_design_v0.md`（7人の人格・帯別コメントの正本）の増産巻。実在賞レースの審査講評をリサーチ・分析し、型を抽出して量産する。**実在審査員の口調・言い回しの模倣はしない**（judge_design §10-D の権利線引きに従う。抽出するのは「講評の構造」のみ）。固有名は架空、数値は全て【仮】。

## 0. リサーチ分析 — 実在講評は7つの型に分解できる

実在の全国賞レース決勝の審査コメント（近年3大会ぶん）とファンのまとめ・分析記事を調査。講評は以下の型の組み合わせで構成されている:

| 型 | 構造 | 例の骨格（一般化） |
|---|---|---|
| T1 技術指摘型 | 具体箇所+数値/技術用語 | 「ボケ数が〜」「つかみの秒数が〜」「システムの構造が〜」 |
| T2 主観愛型 | 評価軸を放棄して好みを宣言 | 「僕はこういうの大好き」「点数関係なく好き」 |
| T3 比較型 | 過去の同組・前年・1本目との差分 | 「去年より〜」「1本目と2本目で〜」「2本目は100点付けてもいいかなって」 |
| T4 助言型 | 減点理由+改善の方向 | 「〜すればもっと跳ねる」 |
| T5 全肯定型 | 短い断言（最高点とセットで効く） | 「文句なし」「完璧」 |
| T6 愛ある辛口型 | ダメ出し+関係性/期待のフォロー | 「あんたらならもっとできる」 |
| T7 自分語り型 | 審査員自身の経験に引きつける | 「私が現役の頃は〜」 |
| 追加の観察 | **短いコメントは点数との乖離が炎上の火種になる**（実在大会で毎年物議）。辛口は「どこが」を言えば愛になり、言わなければ暴言になる | |

執筆規則（judge_design §10-A を拡張）:
1. 辛口コメントは必ず**T1（具体箇所）を含む**。人格・才能の否定は書かない
2. 優しいコメントは必ず**次の一歩**か**具体的な褒め所**を含む（空の全肯定はA帯のみ）
3. 同じ審査員の中で T1〜T7 の配分を人格に合わせて固定する（例: 天堂寺=T1+T5、花園=T6+T7、卯月=T1+T4、白波=T2+T6、目白=T1+T4、神楽坂=T7+T2、音羽=T2+T3）

## 1. 予選の紙講評 増産（最頻出コンテンツ・通過/敗退×回戦帯）

予選は審査員非表示・紙1行。**プレイヤーが10年で最も多く読む文章**なので最優先で増産する。各帯で重複なしローテ。

### 1〜2回戦（通過）— 「見どころを一つ拾う」
1. 「つかみ良し。次はネタの後半に山をもう一つ。」
2. 「声が会場サイズに合っていた。この調子で。」
3. 「設定の発明あり。磨けば武器になる。」
4. 「ツッコミの語彙が良い。もっと聞きたい。」
5. 「1つ目のボケまでが速い。良い癖です。」
6. 「掛け合いのリズムが既に商品。ネタが追いつけば化ける。」
7. 「客の年齢層を見て間を変えていた。通過。」
8. 「4分間、姿勢が良い。舞台慣れを感じる。」

### 1〜2回戦（敗退）— 「直せる一点」必須
9. 「4分の配分に難。前半に詰め込みすぎ。」
10. 「ボケの手数不足。テンポは良い。」
11. 「二人の関係がネタに乗っていない。」
12. 「設定説明に1分は長い。30秒で入って。」
13. 「ツッコミが全部同じ角度。3種類は欲しい。」
14. 「声は届いたが言葉が届かず。単語を減らして。」
15. 「オチが2分地点で見えた。隠し方の問題、面白さの問題ではない。」
16. 「衣装と立ち位置は既にプロ。中身の熟成待ち。」

### 3回戦〜準々決勝（通過）— 「プロの水準で比較される」
17. 「この回戦で客を3回置きに行った度胸を買う。」
18. 「中盤の畳みかけは今日の全組で最速。」
19. 「教科書通りの構成に一箇所だけ違反があり、そこが一番ウケていた。良い違反。」
20. 「昨年より明確に手数が増えた。継続は評価される。」
21. 「音響なしでこの緩急は技術。次も見たい。」

### 3回戦〜準々決勝（敗退）— 「壁の正体を言語化する」
22. 「面白い。ただ『今年の4分』ではない。新しさが一つ足りない。」
23. 「完成度は通過組と同じ。差は爆発力。安全運転を一箇所捨てて。」
24. 「この回戦からは『上手い』は前提。『お前らだけのもの』を持ってきて。」
25. 「前半で客を掴んだのに後半は維持だけだった。攻め続けて。」
26. 「ネタは決勝級、演者がまだ準々級。体力と場数。」
27. 「2年前と同じ型。型は武器だが、進化しない武器は読まれる。」

### 準決勝（敗退）— 「最も残酷で、最も丁寧に」（決勝到達25〜35%の裏側＝大多数が読む）
28. 「決勝との差は0.5本。ネタが1本ではなく、1本半必要だった。」
29. 「今日の9組と入れ替わっても違和感はない。足りないのは違和感の方だ。」
30. 「完璧な4分。完璧すぎて事件が起きなかった。」
31. 「敗因を探すな。今年は相手が強かった。それだけの年もある。」
32. 「あなたたちを落とすことに審査は30分揉めた。それが今年の到達点。」

## 2. 決勝コメント増産（7人×頻出帯B/C＋厳しめE寄り・優しめA寄りの拡張）

judge_design の人格・口調を維持し、各人に「厳しめ1本・優しめ1本・型に沿った1本」を追加する。

### 天堂寺 銀郎（T1+T5・辛口の門番）
- 厳しめ(D): 「笑いは取れとる。芸で取ったのは前半の2つだけや。あとは勢いの借金。」
- 優しめ(C): 「今日のは60点や。ただな、60点の取り方が正しい。正しい60点は90点に育つ。」
- B: 「ツッコミの一言目、全部半拍で入っとった。あれは稽古やない、年輪や。」

### 花園 千代（T6+T7・愛ある説教）
- 厳しめ(D): 「うちな、あんたらのこと好きやから言うで。今日のは相方に甘えた4分や。一人で背負う場面、一個もなかったやろ。」
- 優しめ(C): 「ようここまで来たなぁ。うちが初めて決勝出た年より、ずっと上手や。……上手やから泣けてくるんよ。」
- B: 「途中でな、ネタ忘れて見入ってもうた。審査員にネタを忘れさせたら勝ちや。」

### 卯月 走太（T1+T4・数字の人）
- 厳しめ(D): 「最初の60秒でボケ4つ、うち3つが同じ構造でした。構造の被りは手数を実質1つに圧縮するんですよ。」
- 優しめ(C): 「点数は伸ばせませんでした。でも帰ってVTRを3回見てください。2分10秒のあの返し、あれが皆さんの鉱脈です。」
- B: 「前半で仕込んだ違和感を後半で全部回収して、しかも回収順が仕込み順の逆。理論値です。」

### 白波 剛（T2+T6・熱の人）
- 厳しめ(D): 「今日のお前らは『ウケたい』が先に立ってた！ 客はな、『言いたいことがある奴』を笑うんだよ！」
- 優しめ(C): 「点は辛くつけた。けどよ、最後のツッコミの声、あれは本物だ。あの声をネタの頭から出せた日、お前らは決勝の常連だ。」
- B: 「途中から審査を忘れて客として笑ってた。減点材料はそのあと思い出して探した。見つからなかったよ！」

### 目白 慧（T1+T4・静かな設計者）
- 厳しめ(D): 「4分間に感情の起伏が1回でした。設計上は3回置けます。図面から引き直すことをお勧めします。」
- 優しめ(C): 「粗は5箇所数えました。ただ、5箇所とも『挑戦した結果の粗』です。守りの完璧より攻めの粗を評価します。」
- B: 「1本目のネタでこの完成度を出して、2本目の余力も感じさせる。年間の設計ができている組は強い。」

### 神楽坂 とんぼ（T7+T2・飄々の目利き）
- 厳しめ(D): 「うちの師匠がよく言ってました。『客を笑わすな、客が笑うのを邪魔するな』。今日は少々、邪魔してましたな。」
- 優しめ(C): 「点数ですか？ 勘です。でもね、私の勘は今日、あなたたちの来年を見ました。」
- B: 「間がいいとか悪いとかじゃなくて、あなたたちの『時間』が流れてた。ああなったら芸は勝ちです。」

### 音羽 ルリ（T2+T3・客席の代弁者）
- 厳しめ(D): 「ごめん、はっきり言うね。テレビの前の人、2分でチャンネル変えてたと思う。劇場の笑いとお茶の間の笑い、今日は劇場だけだった。」
- 優しめ(C): 「点数以上に大事なこと言うね。私、明日あなたたちのコンビ名を3人に話す。それって点数より強いよ。」
- B: 「去年より圧倒的に良くなってる。成長曲線ごと見せられると、私たちってもう応援しかできないのよ。」

## 3. 敗者復活の実況・観客コメント（会場の外の声）

敗者復活は審査員でなく**会場外の観客と実況**が声の主役（実在の形式に準拠しつつ架空化）:

- 実況（汎用）: 「さあ、運命の1枠。44組の今年が、いま終わり、1組の今年が続きます。」
- 実況（通過発表）: 「——決まりました。〈コンビ名〉！ 夜の決勝、最後の椅子です！」
- 観客の声（プール・ランダム2本表示）: 「昼のネタ、一番笑ったのあの組だった」「寒い中並んだ甲斐あった」「来年は最初から決勝で見たい」「隣の人、ずっと泣いてた」
- 落選側の帰り際（実況の小声・演出用）: 「……敗者復活戦、という名前ですが、ここで敗者になる組が44組いる。それでも来年、また44組がここに立つんです。」

## 4. 票割れ演出（judge_design §11-1 未決への提案・全て【仮】）

決勝で審査員の表示点が割れた時（最高-最低≥6点）だけ発生する掛け合い1往復:

- 天堂寺×音羽: 音羽「え、渋くない？」天堂寺「渋いのが仕事や。」
- 卯月×神楽坂: 卯月「理論的には減点なんですよ」神楽坂「私は勘で加点です。困りましたな。」
- 白波×目白: 白波「点数じゃねえんだよこういうのは！」目白「点数をつける仕事です。」
- 実装: 掛け合いは1組の審査結果につき最大1回・同じ組み合わせは1大会1回。**判定には無関係（表示演出のみ）**の原則を維持

## 5. 実装メモ

- 全て judge_design §0 の仕組み（帯×審査員×条件フック・直近使用ID回避）に載る。新機構は§4の「票割れ検知（表示点の最大差≥6）」のみ
- 予選紙講評は§1で通過13本・敗退19本の計32本体制（従来7本）。回戦帯でプールを分けるのが v1 の変更点
- 文言の禁則: 人格否定・容姿言及・「センスがない」等の才能否定は全帯で禁止（10-A）。辛口は必ず場所を指す
