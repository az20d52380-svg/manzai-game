# ネタカタログ v0 — 漫才の種類リサーチと題材リスト量産

- 日付: 2026-07-04
- 位置づけ: ネタ資産システム（`neta_system_design_v0.md`）の「型」の裏付けリサーチと、ネタ名自動生成用の題材リスト量産（同 §8-3 の宿題）。固有名は架空・題材は一般名詞のみ（実在コンビの代表ネタ題材と偶然一致しても問題ない普遍語に限定）。

## 1. リサーチ — 実在の漫才分類

芸人向けネタ講座・漫才分析サイト・百科事典を調査。実在の分類は概ね:

1. **しゃべくり漫才**: 漫才師本人としての対話だけで進める正統派。賞レースで最も評価されやすいとされる
2. **コント漫才（漫才コント）**: 導入は本人、途中から「やらせてくれ」で役に入る。センターマイク1本・小道具なし
3. **システム漫才**: 特定のフォーマット（構造の反復）を最後まで使い続ける型。近年の賞レースの主流の一つ
4. 派生・下位分類: キャラ漫才（しゃべくり派生）／リズム・音ネタ漫才／大喜利系・ギャグ系の展開／ノリボケ・Wボケ・ツッコミ不在などの役割変形

### ゲームの「型5種」との対応（neta_system §2 の型を実在分類で裏打ち）

| ゲーム内の型 | 実在対応 | 特性の意図 |
|---|---|---|
| 王道しゃべくり | しゃべくり漫才 | 基準型。減点されにくく爆発もしにくい |
| つかみ型 | 出だし特化の構成 | 序盤ウケ重視。短い持ち時間の予選に強い【仮】 |
| 後半山型 | コント漫才に多い後半爆発構成 | 決勝の長尺で映える【仮】 |
| 伏線回収型 | システム漫才・構成派 | ハマれば大爆発、構成がバレると失速 |
| 飛び道具 | リズム・キャラ・音ネタ系 | 分散大。神楽坂が愛し天堂寺が渋い（judge連動） |

- 【将来拡張】「漫才コント」と「しゃべくり」の二大区分を型の上位属性として持たせる案（型5種 × 上位2区分 = 実質10種）。フェーズ2の審査員相性を厚くできるが、MVP〜周回版フェーズ1では型5種のみで十分

## 2. 題材リスト（自動生成用・100本）

## 3. 生成規則【仮】

1. 新ネタ作成時: 題材はカテゴリ均等の重み付きランダム3候補からプレイヤーが選択（or おまかせ）。型はプレイヤー選択（neta_system §3）
2. 同一キャリア内の題材重複は禁止（100本×持ちネタ4枠なら枯渇しない）
3. 表示名の揺らぎ: 「〇〇」「〇〇のやつ」「新・〇〇」「〇〇2026」等の接尾辞プールで、同題材でも周回間の見た目が変わる【仮】
4. 谷口（相方）の題材コメント: カテゴリ別に1行ずつ用意（例: C職業系→「俺、警官の顔だけは自信あるで」）——顔合わせ演出と接続
5. 権利ガード: 題材リストへの追加は「一般名詞・状況語のみ」「実在コンビ名・番組名・特定のフレーズを含まない」をレビュー基準にする
