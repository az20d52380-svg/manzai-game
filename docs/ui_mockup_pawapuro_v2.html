これは何 / 直せる所

パワプロ サクセス／ウマ娘のUI定石に寄せた仮モックです（ブラウザで動く・配色/キャラ/アイコンは全部【仮】）。丸ゴシック＋ポップ配色＋キャラ＋練習タイルで「育成ゲームの画面」に。

一点だけ設計の分かれ道：パワプロは「伸びる能力・経験点」を見せる派ですが、今のゲーム設計は効果を隠す派でした。モックはパワプロに寄せて“どの能力が伸びるか”を色ドットで表示しています（数値は伏せ）。ここは好みで選べます。

この方向でOK？ OKならSwiftUIに移植して1年MVPに配線（ビルド＆実機はMac）。直すなら「もっと明るく／キャラの絵が欲しい／色を変えて／能力は隠して」等ひと言で即やり直します。SCREEN 01のコマンドは実際に押せます。
