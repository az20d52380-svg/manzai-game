# 固定ライバル設計 v0 — 4組の経歴時間軸と「時代」の演出

- 日付: 2026-07-04
- 位置づけ: 拡張アイデア集v1.2 A-3の具体化。作り込みの架空強豪4組に**経歴の時間軸**を持たせ、大会の顔ぶれが年々入れ替わる「時代」を作る。固有名は全て架空（実在コンビを連想させる名前・芸風の特定的な組み合わせは避ける）。数値は全て【仮】。
- リサーチ根拠: (a)育成ゲームの名ライバル像＝「天才でありながら努力も惜しまない」「最初は嫌味、作品を重ねて関係が変わる」——**関係の変化そのものがコンテンツ** (b)実在賞レースのファン文化はSNSで推しコンビを追う（顔ぶれの物語を皆が覚えている） (c)実力者が準決止まりを繰り返す・常連が急に消える・数年ぶりに帰ってくる、が実際の大会の物語資産

## 0. 機構の大原則 —「並ぶが、戦わない」

本作の判定は絶対評価（ライン方式）であり、実在の予選と同じ構造。ライバルとは**直接対決しない**。「同じ大会に出て、別々に採点され、結果として顔ぶれが並ぶ」——結果発表の貼り紙に自分の名前とライバルの名前が並ぶ/並ばないことが、そのまま物語になる。

- ライバルの成績は**シナリオ駆動**（年表＋周回ごとの揺らぎ）であり、シミュレーションしない（バランスへの影響ゼロ・演出層のみ）
- 例外はただ一つ: **優勝枠**。自分が決勝で敗れた年の優勝者としてライバル名を表示する（選出仕様は finals_direction_v0.md §4-D で確定。コード実装後に「別のコンビだった」を差し替え）。ライバルの誰かが勇退済みなら汎用モブ名

## 1. ライバル4組

### ① 金字塔（きんじとう）——「先に行く同期」

- 天才×努力の正統派しゃべくり。養成所の同期で、最初の楽屋で「君ら、まだその段階？」と言い放つ嫌味な男たち（名ライバルの文法: 嫌味から入って、関係が深まる）
- **年表【仮】**: 1年目=2回戦 / 2年目=準々 / 3年目=準決 / 4年目=決勝(敗退) / 5年目=決勝(敗退) / **6年目=優勝・勇退** / 以降は審査側・テレビの人として時々登場
- 役割: 初回プレイの「先に行かれる」痛み。プレイヤーの初優勝が7年目前後に来る設計なので、**その1年前に同期が頂点を取って去る**。勇退後、廃業した劇場の楽屋で「……お前らのほうが、俺たちより漫才好きだったよ」と一言残す
- 関係イベント: 2回戦の楽屋（嫌味）→ 準決の袖（無言の会釈）→ 優勝後の祝辞（本音）の3段階

### ② 夜行列車（やこうれっしゃ）——「終列車のベテラン」

- 結成12年目からゲーム開始時点の年表に登場する苦労人コンビ。プレイヤーの3年目に**出場資格ラストイヤー**（グランプリ＝結成15年以内【仮・実在準拠】）を迎える
- **年表【仮】**: 1年目=準決 / 2年目=準々 / 3年目=ラストイヤーで準決まで→敗者復活で散る / 4年目以降=ベテラン大会ルート（roadmap_v1.md）に移籍し、そこで優勝する
- 役割: 資格期限という時計の存在をプレイヤーに教える「予告編」。敗者復活で散る夜、出待ちのファンに深々と頭を下げる姿を見せる（このゲームで一番泣かせに行く脇役）
- 関係イベント: 大阪遠征の夜行バスで隣り合わせる（タイトル回収）

### ③ ミラーボール兄弟——「バズって消えて、帰ってくる」

- 華とSNS適性の塊の飛び道具コンビ。切り抜きでブレイク→タレント化してコアを失う、fanbase設計（agency_fanbase_design_v0.md）の生き見本
- **年表【仮】**: 1年目=3回戦 / 2年目=準決＋SNSバズ / 3年目=冠番組を持ち賞レース欠場（タレント化） / 4〜5年目=1回戦から出直すも2回戦落ち（飽きられ） / 6年目=単独ライブで劇場に帰る / **7年目=決勝返り咲き**（返り咲きの実在感: 史上2組の偉業として演出）
- 役割: 「知名度とコアは別物」をプレイヤーの目の前で実演する教材。彼らの凋落と帰還のニュースが、冠番組（agency_fanbase§3）を選ぶ前の警告として機能する
- 関係イベント: バズ期に「お前らも動画やれって」→ 凋落期に「……劇場、いいよな」

### ④ 静物画（せいぶつが）——「準決勝の魔物に愛された画家」

- コアなお笑いファンが全員好きな技巧派。**準決勝常連なのに決勝に行けない**（実在で語られる「コアに刺さりすぎて一般客の決勝で伸びない」現象の擬人化）
- **年表【仮】**: 1〜8年目=毎年準決勝で敗退（8年連続・SNSでは毎年「なぜ落ちた」の祭り） / 9年目=ネタの型を変えて**初の決勝**（飽きられ解除の実演） / 10年目=資格ラストイヤー・結果は周回ごとに変動（決勝で優勝する周回もある）
- 役割: 「実力≠決勝」の世界の深さ。プレイヤーが準決で落ちた年、貼り紙の前で静物画の二人も落ちている——「あの静物画でも落ちるのか」という慰めと畏れ
- 関係イベント: 準決の楽屋で毎年同じ自販機の前に立っている。9年目、初めて缶コーヒーを2本買って渡してくる

## 2. 周回との関係

- 年表は「基準シナリオ」であり、周回ごとに±1年の揺らぎと結果分岐（静物画の10年目など）を持つ——2周目以降の既視感対策
- プレイヤーの成績はライバル年表に干渉しない（MVP〜中期）。将来「自分が優勝した年は金字塔の優勝が消える」等の相互作用を入れる場合は、優勝枠の整合だけ special-case で処理
- ライバルの敗退・引退・返り咲きは年次リザルトの「今年の業界ニュース」欄で必ず1行報じる（世界が動いている感）

## 3. 実装コスト

- 本体は年表データ（4組×10年×結果）＋演出テキスト（各組8〜12本）＋結果発表画面への名前差し込み。判定ロジックへの変更はゼロ
- 立ち絵は各組1枚＋表情差分2で開始可能（H-2のデフォルメ方針）
