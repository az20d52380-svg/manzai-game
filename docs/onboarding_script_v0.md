## 2. 第1〜8週の導線台本

## 5. 実装メモ

- 固定イベント（第4/5/8/12/30/47週）以外は全て条件差し込み——「教えることは1週に1つ」の総量規制を守る
- 初回のみのスロー演出（差分ポップ0.2秒遅延等）は2周目以降すべてカット（骨子§6-3の省略仕様に接続）
- 支配人の続投は人物帳（dialogue_batch3 §2）に追加登録して管理する（未決だった§6-2を「続投する」で確定提案——年1回・GP初戦前に一言だけ登場【仮】）
- 紙講評の初回固定（§3-3）はチュートリアルのみの特例。以降は通常プール
