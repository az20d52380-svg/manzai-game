# グランプリ決勝 演出フロー台本 v0

- 日付: 2026-07-03
- 位置づけ: 第47週（決勝・敗者復活）の画面演出のカット割り台本。判定は既存のGameCoreロジック（内部スコアvsライン）のまま、**見せる順番と間**だけで最大の感情を作る。固有名は架空、数値は全て【仮】。
- 演出原則:
  1. **結果の1秒前が最も面白い**（improvement_proposals §4）。開示は常に「溜め→開示」
  2. 初回プレイの主役は敗北。**負けの演出に勝ちと同じ物量をかける**
  3. スキップ可。ただし初回の決勝だけはスキップ不可の価値がある（要プレイテスト）

## 0. 週開始時の分岐

| 状態 | ルート |
|---|---|
| 準決勝通過済み | A. 決勝ルート（§2） |
| 準決勝敗退（敗者復活対象） | B. 敗者復活ルート（§1）→通過ならAへ合流 |
| どちらでもない | 通常週（テレビの前で決勝を見る演出1カット→「来年は、あっち側だ」） |

## 1. ルートB: 敗者復活戦（昼・屋外）

1. **カット1（空）**: 冬の屋外ステージ。白い息。テロップ「敗者復活戦」
2. **カット2（会話）**: dialogue_design §4「昼の結果は忘れろ」→ 谷口「忘れられるか。……忘れんまま勝つんよ」
3. **カット3（出番）**: 出囃子→暗転→**客席の笑い波形メーター**が走る（ネタ本編は見せない。波形と歓声SEで語る）
4. **カット4（開示・溜め2秒）**: 「復活の1組は——」
   - **通過（実測0.2〜5%の奇跡）**: 画面が夜の決勝会場へワイプ。移動する車内の無言の二人→「行くぞ」。実績「敗者復活の奇跡」解放演出
   - **敗退（通常）**: 波形メーターが良くても届かない残酷さを1カットで。「悔しさは、来年の一本目に書く」→年末パートへ
5. 設計意図: 初回はキャリア中4〜5回立つ舞台（実測）。**毎年の風物詩として短く様式化**し、通った年だけ演出が爆発する

## 2. ルートA: 決勝（夜・大劇場）

### 2-1. 入場〜ネタ

1. **カット1**: 楽屋の鏡前。ネクタイ/衣装を直す手元のアップ。相性15以上なら谷口が無言で襟を直してくる
2. **カット2**: 袖からセンターマイクまでの歩き（実在の決勝の象徴的構図）。歓声が波のように
3. **カット3**: ネタ（波形メーター演出・約6秒）。ネタ資産システム導入後はここでネタ名テロップ（「勝負ネタ:『留守番電話』」）

### 2-2. 審査員採点（judge_design_v0.md §0と接続）

1. 7人の審査員が順に点数開示。**開示順は固定**: 音羽→白波→卯月→花園→目白→神楽坂→**天堂寺（トリ）**
   - ミーハー(音羽)から始まり、辛口の権威(天堂寺)で締める——点数が上がっていく/落ちる緊張の設計
2. 各開示は「表情1秒→点数→一言コメント」。コメントは帯×フックで選定済みのもの
3. 5人目終了時に**中間集計テロップ**「優勝ラインまで、あと○点」——内部判定は済んでいるが、表示上の攻防を演出
4. 天堂寺の点数だけ、めくれるまでに+1秒の溜め

### 2-3. 結果開示

- **優勝（内部スコア≥95）**:
  1. 全点数の合計が優勝ラインを越える瞬間に紙吹雪・暗転なしの生放送感
  2. 泣き崩れる谷口/呆然の主人公（相性帯で芝居が変わる: 阿吽帯なら二人とも笑ってしまう）
  3. dialogue_design §4の耳打ち「……なあ、腹減ったな」
  4. **エピローグ（勇退モンタージュ・研究反映）**: 翌日から半年、スケジュール帳が黒く埋まる紙芝居→新幹線の車窓×4季節→最後のカット「10年前と同じファミレスの同じ席」→年表画面（improvement §1）→スタッフロール
- **敗北（決勝到達・初回の17〜28%が見る画面）**:
  1. 優勝者名の開示は**他コンビ名を出さずに**「優勝は——別のコンビだった。」の一文＋照明が落ちる引きの画
  2. 袖での会話: 谷口「来年の、いっちばん面白いネタの話、していい？」
  3. リザルト地の文「決勝の照明は、思っていたより熱かった。あの熱を知らないまま終わらなくて、よかった」
  4. 実績「決勝の照明」解放→**才能解放演出をこの直後に置く**（敗北の夜に、次への火を灯す順序が周回動機の設計核心）

## 3. 音・触覚メモ

- 波形メーター中はBGM無し（客の笑い声SEのみ）。点数開示は無音→スタンプ音
- 天堂寺の開示のみ、会場SEも消す（伝説の間）
- ハプティクス: 点数開示ごとに弱、優勝瞬間のみ長強

## 4. 実装コスト注記

- 全カット静止画＋テキスト＋SEで成立する構成にしてある（アニメ不要）。波形メーターのみ簡易アニメ
- 審査員の表情は各3種（開示前/高得点/低得点）×7人=21枚が最小セット

## 4-B. 【2026-07-04追記】最終決戦（TOP3方式）の導入方針

実在の決勝は「10組が1本目→上位3組が2本目→審査員が点数でなく**3組から1組を選ぶ投票**」の二段構造。本作の現状は決勝1判定（ライン91）でこの構造を持っていない。導入は2段階で行う:

1. **フェーズ1（演出のみ・バランス不変）**: 判定は従来どおり内部スコアvs91の1回。ただし見せ方を二段にする——スコアが89以上【仮】なら「最終決戦進出（TOP3）」演出を挟み、91以上で優勝。開示は点数でなく**7人が1組ずつ札を上げる投票演出**（スコアと優勝ラインの差から票数を生成: 圧勝=6〜7票、接戦=4票、敗北=1〜3票、**大差負け=0票**【仮・2026-07-04オーナー指示で0票を追加。「1票も入らなかった夜」は屈辱でなく再起の名場面として演出する】）。「TOP3に入って1票差で敗れる」が初回プレイ最大の悔しさ装置になる（0pt称号「あと一票」「白票の夜」新設候補）
2. **フェーズ2（ネタ資産システムと同時・判定も二段化）**: 1本目でTOP3判定（ライン89【仮】）→**2本目のネタ**で優勝判定（ライン91）。neta_system_design §4の決勝2本制と完全に噛み合う（「2本目がなかった」という実在型の敗因が生まれる）。導入時は優勝率の再計測必須
