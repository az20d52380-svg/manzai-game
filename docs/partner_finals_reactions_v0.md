- 日付: 2026-07-05
- 位置づけ: 決勝敗北の結果発表で流れる**主人公の相方の袖反応**を、相方ごとの声に差し替える差分集。`relationship_champions_v0.md §4`（谷口版が基準）の**袖行のみ**を各相方の声で置換する。フレーバー行（「優勝は——〈N〉。」）＝**中立ナレーター**は差し替え対象外（`champion_selection §0-3`）。
- 声の出典: `partner_characters_v0.md`（早乙女／二階堂／八雲の口調・二面性・口癖）。判定ロジック・golden・バランス数値には触れない（演出層・表示分岐フラグのみ）。

- **関西弁は"全員のデフォルト"にしない**（オーナー指示の芯）。谷口＝関西弁は谷口のキャラ。**早乙女・八雲は標準語**で、関西弁を一切使わない。
- ★**二階堂だけは例外的に関西弁**。理由: `partner_characters §2` で「大阪から単身上京・口癖『知らんけど』」＝関西弁が本人の設定の芯であり、谷口のコピーではなく二階堂自身の声。これは「関西弁を全員に付ける」こととは別（各キャラ固有の声を守っているだけ）。
  - **【オーナー切替ポイント】** 「二階堂も標準語にしたい」場合は本書 §3 のみを標準語へ差し替えれば済む（他は不変）。`partner_characters §2` の口癖「知らんけど」・大阪設定も併せて要改訂になる旨だけ注記（キャラの芯が変わるため）。
- 役柄側（同期・後輩・先輩）の声は相方に依らず不変（`champion_selection §0-3`）。本書が差し替えるのは**主人公の相方の袖反応だけ**。

## 4. 実装メモ

1. **差し替え単位**: `relationship_champions §4` の各セルの「谷口（袖）:」行を、選択中の相方IDで本書の対応行に差し替える。フレーバー行・役柄セリフ・一人称地の文の三人称ナレーション部は不変（`champion_selection §0-3` の声帰属に従う）。
2. **決定性・非干渉**: 差し替えは相方IDによる**表示分岐のみ**（`runSeed`・優勝者選出・golden に非干渉）。相方IDはセーブ済みのゲーム状態から読む。
3. **未実装相方のフォールバック**: 5人目以降の相方が追加されるまでは、未定義相方は谷口版（関西弁）ではなく**八雲版（標準語・中立寄り）を汎用フォールバック**にする（関西弁を誤って初期値にしないため・オーナー指示の芯）。
4. **絆連動（任意・`relationship_champions §4`）**: A/B/C案の深化は相方版では当面A/B相当の1行のみ（本書）。絆高のC案相当は今後の拡張【仮】。
