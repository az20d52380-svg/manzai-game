# 会話・掛け合い設計 v0 — 相方「谷口」と週次フレーバー

- 日付: 2026-07-03
- 位置づけ: 週次行動の結果画面・大会前後・リザルトに載せる会話テキストのトーン定義とサンプル集。人名は架空。数値は全て【仮】。
- 設計原則:
  1. **会話はご褒美であって義務にしない**（1画面2〜4行・スキップ可・選択肢はイベント時のみ）
  2. **相性値で同じ場面の会話が変わる**——数値の成長を「関係の変化」として見せる。これが相性というパラメータの感情的な正体
  3. 谷口は「ボケ」だが、会話では**ふとした一言が核心を突く**タイプにする（ネタ中とオフの二面性がコンビものの王道）
  4. 説明台詞禁止。システムの数値は会話で言い直さない（UIが言うことを人間に言わせない）

## 1. キャラクター定義

### 主人公（プレイヤー分身・名前は初回に入力）
- ツッコミ／ネタ作り担当。心配性で計算屋。金勘定と進行管理は全部こっち
- 一人称「俺」。モノローグの語り部（リザルト画面の地の文はこの人物の内心）

### 相方・谷口（たにぐち）
- ボケ担当。楽観・大食い・バイト掛け持ちの達人（居酒屋/引越し/交通量調査）
- 実は繊細で、スベった夜は寝ない。それを絶対に見せない
- 口癖: 「まあ、なんとかなるやろ」「腹減ったな」「それ、ネタになるな」
- 天然に見えて、舞台勘だけは本物——という「相方への信頼」の根拠を随所に置く

### 関係の初期設定
- 高校の同級生。文化祭の漫才が原点。上京して同じアパートの隣室

## 2. 相性帯によるトーン変化（同一場面の書き分け見本）

場面: 稽古（ネタ合わせ）終わり

- **相性 0〜7（他人行儀）**
  - 谷口「……今日、こんなもんかな」
  - 俺「ああ。……じゃあ、また明日」
  - *（沈黙が長い。まだ、お互いの笑いのツボを探り合っている）*
- **相性 8〜14（噛み合い始め）**
  - 谷口「3つ目のボケ、順番変えたら跳ねる気がすんねんな」
  - 俺「俺も同じとこ引っかかってた。……先言うなよ」
  - 谷口「それ、ネタになるな」
- **相性 15〜20（ツーカー）**
  - 谷口「なあ、あそこさ——」
  - 俺「変えた。もう直してある」
  - 谷口「……こわ。俺ら、そのうち喋らんでよくなるで」
- **相性 21+（阿吽・相方ガチャ上位帯）**
  - *（谷口が目配せ。俺が頷く。修正箇所の共有、以上）*
  - 谷口「腹減ったな」
  - 俺「それは口に出すんかい」

## 3. 週次行動のフレーバー会話（各3種ローテ＋条件差し替え）

### 稽古「ネタ作り」
1. 谷口「お題ちょうだい。なんでもボケるから」／俺「じゃあ『留守番電話』」／谷口「……もしもし？」／俺「出るな。留守番の意味よ」
2. 俺「昨日の客、どこで笑ったか全部メモしてある」／谷口「お前のそういうとこ、俺は好きやで。モテへんと思うけど」
3. （所持金<5万時）谷口「腹減って集中でけへん」／俺「ネタ帳食う勢いでやれ」／谷口「紙は流石に……いや、天ぷらにしたら？」／俺「書け」

### 稽古「舞台稽古」（有料）
1. 俺「8万。今月の稽古場代」／谷口「俺らの4分、時給に直したらエグいことなってんな」／俺「だから今日で回収すんの」
2. 谷口「鏡の前でやると、俺のボケ、思ったより顔うるさいな」／俺「客席からは、それがちょうどいい」

### バイト「キツい」
1. 谷口「引越し屋のバイト、今日で通算タンス200棹目」／俺「その数え方やめろ。悲しくなる」
2. （3週連続時）俺「……手、マメだらけだな」／谷口「センターマイク握るための予行演習よ」

### 休む「完全休養」
1. 谷口「今日は寝る。プロとして寝る」／俺「その言い方だと寝るのが本業みたいだろ」
2. （体力<20時）*久しぶりに夢も見ずに眠った。起きたら昼で、少し泣きそうになるくらい体が軽かった。*

### 休む「相方と過ごす」
1. 銭湯。壁のペンキ絵を見ながら谷口が言う。「売れたら、こういう仕事もすんのかな」／「ペンキ屋を？」／「営業の話よ」
2. （相性15以上）ファミレス。ネタの話を一度もしなかった。それでいて、帰り道に二人ともネタ帳を開いた

### オファー「お金重視」
1. 谷口「ギャラ聞いた？ 俺、ゼロ一個数え間違えたかと思った」／俺「営業スマイルの練習しとけ」／谷口「（満面）」／俺「本番でそれ出すな」

## 4. 大会の前後

### 大会前夜（共通・メンタル帯で差し替え）
- メンタル50以上: 谷口「緊張しとる？」／俺「してない」／谷口「嘘つけ。……俺はしてる。半分持って」／俺「緊張って割れんのよ」
- メンタル40未満: 眠れない。壁の向こうから、隣室の谷口が小声でネタを回す声がする。……あいつも起きてる。それだけで、少し眠れる気がした

### 予選通過（汎用）
- 谷口「名前あった！ 俺らの名前あった！」／俺「落ち着け。まだ○回戦」／谷口「お前こそ、その紙持つ手ぇ震えてんぞ」

### 予選敗退（汎用→年数フック）
- 通常: 帰りの電車は静かだった。谷口が先に口を開く。「……来年の話、していい？」／それが許される距離まで、黙って二駅かかった
- 同回戦3年連続: 谷口「3年連続、同じ壁。……これ、壁の方が俺らに慣れてきてんちゃう？」／俺「なら、慣れられてるうちに越えるぞ」

### 準決勝敗退・敗者復活へ
- 俺「昼の結果は忘れろ。夜にもう一回ある」／谷口「忘れられるか。……忘れんまま勝つんよ」

### 決勝進出決定
- 谷口はしばらく黙っていた。それから、いつもの顔で言った。「腹減ったな」／「……そうだな」／泣きそうな時にメシの話をするのが、こいつの優しさだと知っている

### 決勝敗退（初回プレイの山場・ここに最大の筆を割く）
- 照明が落ちた舞台袖。谷口「なあ」／俺「なんだ」／谷口「来年の、いっちばん面白いネタの話、していい？」／俺「……ああ。聞かせろ」
- *（リザルトへ: 「決勝の照明は、思っていたより熱かった。あの熱を知らないまま終わらなくて、よかった」）*

### 優勝（勇退エンド）
- 紙吹雪の中、谷口が耳打ちしてきた。本人は名言のつもりだったと思う。「……なあ、腹減ったな」／「俺もだよ。……行くか、あのファミレス」／10年前と同じ席で、ドリンクバーだけ頼んで、朝まで笑った

## 5. 年次リザルトのモノローグ（俺の独白・年の成績で分岐）

- 躍進年: 「去年より二段深いところまで来た。空気が薄い。……いい薄さだ。」
- 停滞年: 「今年を『何もなかった年』と書くのは簡単だ。でもタンス200棹と、新ネタ4本は、確かにあった。」
- 貧乏年（最低所持金<0）: 「通帳の残高より、ネタ帳のページが増えた年だった。どっちが財産かは、まだ分からない。」
- 9年目（未優勝）: 「あと1年、と思う。『あと1年』を9回言った人間の『あと1年』は、たぶん重い。」

## 6. 実装メモ

- 会話データは `(場面ID, 相性帯, 条件フック, 重み)` のテーブル選択。GameCoreは場面IDと状態を渡すだけ（テキストはUI層リソース）
- 同一場面の直近使用IDを避けるのは審査員コメントと同じ仕組みを共用
- 谷口の口癖「腹減ったな」は**乱用禁止**（登場は1年に2回まで、うち1回は決勝進出/優勝の感情最大点に置く）。口癖は希釈すると死ぬ
