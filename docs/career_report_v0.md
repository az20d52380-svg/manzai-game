# 10年キャリア検証レポート v0（本番想定・理想形バランス）

- 日付: 2026-07-03
- 方法: `tools/sim_career.py`。1年48週×最大10年、能力・お金・知名度・相性は年またぎ引き継ぎ（体力のみ年初全回復）。グランプリ初優勝でそのキャリアは勇退・終了。各設定2000キャリア、シード20260704で再現可能。週次の行動・数式は `tools/balance_sim.py` を無変更でimport流用。
- ボット: 既存のバランス型・調整型を流用。ただしキャリア用の最小補正として「主効果が上限100の稽古を選んだら、最も低い能力の稽古に差し替え」を追加（補正なしだと3年目以降の稽古週が全部空振りになり実力値57〜66で頭打ち＝ボットの欠陥が実験を埋める。詳細はコード内コメント）。
- 位置づけ: 相方ガチャ・トロフィー周回ボーナス・アイテムは未設計。システムは実装せず、パラメータとして振って「適正な強さ」の材料を採る。**数値は全て【仮】**。

## 3. 結果サマリ（初優勝年の中央値 / 優勝なし率、2000キャリア/設定）

| 相性 | バランス型 中央値 / 3年目までに優勝 | 調整型 中央値 / 3年目までに優勝 |
|---|---|---|
| 5 | 5年目 / 0.1% | 5年目 / 0.0% |
| 10 | 5年目 / 0.9% | 5年目 / 0.0% |
| 15 | 5年目 / 6.4% | 4年目 / 1.9% |
| 20 | 4年目 / 20.6% | 4年目 / 15.2% |

優勝なし率は全設定0%。相性+15（5→20）で中央値が約1年縮み、3年目の早期優勝率が0.1%→20.6%に跳ねる。

### 実験B: 初期能力 一律（相性5・成長あり）→ 周回ボーナスの適正量の材料

| 10（基準） | 4年目 / 17.4% | 4年目 / 3.3% |
| 15 | 4年目 / 28.7% | 4年目 / 18.4% |
| 20 | 4年目 / 40.8% | 4年目 / 41.8% |

優勝なし率は全設定0%。初期能力+5ごとに3年目早期優勝率が約12〜20ptずつ増え、+15で中央値が1年縮む。

## 4. 必須項目の判定

### (1) 初優勝中央値は仕様想定6〜9年に収まるか → **収まらない（4〜5年目・早すぎ）**
基準設定で中央値4年目、最も遅い設定（相性5固定・成長なし）でも5年目。さらに分布が3〜6年目に密集し優勝なし率0%＝「10年やっても届かないコンビ」が存在しない。10年ゲームとして後半5年が空になる。

### (2) 1年目優勝は低確率（数%以下）で起きるか → **数%以下は満たすが、事実上起きない（全設定0.0%）**
2000キャリア×18設定で1年目優勝は1件もなし（2年目も基準設定で0%、最速は初期能力25の7.4%）。「新人が奇跡の初年度制覇」というドラマは現行数値では発生しない。ブレ幅（最大±20）に対しラインの階段（1年目の実力値30前後 vs 準決勝75）が高すぎるため。天才プレイの上振れを許すならブレ幅上限かラインの再設計が必要。

### (3) 多段階化で初優勝が遅くなりすぎていないか → **なっていない（むしろ止められていない）**
7段階に増やしても、成長が速すぎて4年目には準決勝〜決勝ラインを常時超える。多段階はむしろ「毎年どこまで行けたか」の進捗表示として機能しており、初優勝時期の制御はライン側でなく成長側で行うべきと分かった。

### (4) どのパラメータが初優勝年に最も効くか → **稽古による成長速度（実験外の固定要素）が支配的。振った中では初期能力＞相性**
相性+15も初期能力+15も「中央値を約1年」しか動かせない。理由は、週+3〜6の稽古成長が3〜4年で能力を上限付近まで運び、初期差・相性差を飲み込むため。振った範囲では初期能力の方が3年目早期優勝率への寄与がやや大きい（同+15で+39pt vs +20pt、調整型）。つまり相方ガチャ・周回ボーナスは現行成長速度の下では「1年早くなる程度」の緩い加速装置にしかならない。

### (5) 想定とズレている場合の修正候補
初優勝を6〜9年へ遅らせ、分布を広げるためのレバー（優先順）:
1. **稽古の成長逓減**: 能力が高いほど上昇量を減らす（例: 上昇量×(1−現在値/120)）。上限到達を4年→8年前後に伸ばす本命。実装時はSwift側と数式同期（CLAUDE.mdルール5）。
2. **準決勝・決勝ラインの引き上げ＋ブレ幅の下限確保**: 現行はメンタル100でブレ±5となり、実力が閾値を超えた瞬間ほぼ確定通過になる。決勝90前後＋ブレ下限±8〜10で「届いてからも数年勝てない」を作れる。
3. **金のシンク追加（アイテム等の規模感）**: 3年目以降は年+400〜700万ペースで余る。年200〜500万級の消費先（高額アイテム・劇場・弟子など）を置いても破産しない。逆に1〜2年目は数十万しか浮かないので、序盤向けの安価帯も必要。
4. **相方ガチャの設計指針**: 相性固定値は5→20で中央値1年短縮・3年目優勝率+20pt。「良い相方」を相性+5〜10相当にすれば体感差はあるがバランス破壊はしない。+15超は早期優勝券に近づく（ただし1.で成長を遅らせた後は効きが強まるはずなので再計測必須）。
5. **周回ボーナスの設計指針**: 初期能力+5で3年目優勝率が約+12〜20pt、+15で中央値1年短縮。周回1回あたり+3〜5、累計上限+15程度が現行数値での安全圏。これも1.の後に再計測。

## 5. 既知の限界・次の実験

- ボットの最小補正（カンスト稽古の差し替え）を入れており、人間の下手なプレイより最適寄り。初優勝はここより遅くなる余地がある。
- 突発イベント・相方ガチャ本体・アイテムは未実装。成長逓減（上記1.）を入れた版で実験A/Bを再実施するのが次の一手。
- 敗者復活（ライン80）はほぼ通過者を出さない飾りになっている可能性がある。通過ログの計測を次版で追加したい。
