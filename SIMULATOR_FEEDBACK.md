# SIMULATOR_FEEDBACK — 実機/simulatorテスト中の捨てメモ

Claudeのセッションが無いときに書く用。**フォーマットは気にしない**。次にセッションが開いたら、この中身を渡せば分類・proposals化・修正判断はこちらでやる。書いたら下から追記していくだけでいい。

## 書き方（最低限これだけ）
- どの画面/状態で（例: S5週メイン・3年目・大会2週前）
- 何が起きた/変だったか
- （あれば）再現手順

セリフ・声の違和感は下の書式に寄せると尚良い（`proposals/HANDOFF_voice-to-real-device.md` Part C準拠）:
```
スロット: default（平常）
本文: 「（実際の文言）」
状況: 3年目・相性15・大会2週前
感じたこと: 谷口の台詞っぽいのに標準語で喋っていて違和感
```

それ以外（レイアウト崩れ・落ちた・挙動がおかしい等）は自由記述でOK。「なんか寒い」「なんか変」だけでも可——理由が分からなくてもそのまま書く（Part Bの原則8番と同じ：人間の直感を信用する）。

## 画像を渡したいとき
テキストには画像そのものは貼れません（貼っても文字化けするだけ）。**ファイルで保存して、パスをメモに1行書く**とAIが開いて見ます。
1. スクショを撮る（Mac: `⌘⇧4`）
2. `feedback_shots/` フォルダに置く（内容が分かる名前で。例 `s3_stamp_kaburi.png`）
3. メモに `feedback_shots/ファイル名.png` と書き、「赤枠のところ」等の一言を添える
